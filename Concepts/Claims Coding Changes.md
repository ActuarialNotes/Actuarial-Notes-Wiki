[[Actuarial Notes Wiki|Wiki]] / [[Exam 5 (CAS)]] / **Claims Coding Changes**

## Definition
==Claims Coding Changes== refer to modifications in how claims are classified, categorized, or coded in the claims system, which can affect loss development patterns and reserve estimates.

## Examples
- New claims system implementation
- Revised classification codes
- Changed coverage definitions
- Updated claim types
- Different loss categories

## Impact on Reserving
- Historical patterns may not apply
- Development factors may shift
- Comparability issues
- Need to adjust or segment data
