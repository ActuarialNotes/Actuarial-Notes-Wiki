A perpetuity provides for continuous payments. The annual rate of payment at time $t$ is

$$\begin{cases}1, & \text{for } 0 \le t < 10,\\ (1.03)^{t-10}, & \text{for } t \ge 10.\end{cases}$$

Using an annual effective interest rate of 6%, the present value at time $t = 0$ of this perpetuity is $x$.

Calculate $x$.

- A) 27.03
- B) 30.29
- C) 34.83
- D) 38.64
- E) 42.41

## Explanation

The present value for the first 10 years is:
$$\bar{a}_{\overline{10}|} = \frac{1-(1.06)^{-10}}{\ln(1.06)} = 7.58$$

The present value of the payments after $t = 10$ is (using $\delta = \ln(1.06)$ and growth rate $\ln(1.03)$):
$$(1.06)^{-10}\int_0^{\infty}(1.03)^s(1.06)^{-s}\,ds = (1.06)^{-10}\cdot\frac{0.5584}{\ln(1.06)-\ln(1.03)} = 19.45$$

Total present value $x = 7.58 + 19.45 = 27.03$.
