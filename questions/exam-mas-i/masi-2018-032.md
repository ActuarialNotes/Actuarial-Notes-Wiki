Two different data sets were used to construct the four regression models below. The following output was produced from the models:

| Data Set | Model | Dependent variable | Independent variables | Total Sum of Squares | Residual Sum of Squares |
|----------|-------|--------------------|-----------------------|----------------------|-------------------------|
| A | 1 | $Y_A$ | $X_{A1}, X_{A2}$ | 35,930 | 2,823 |
| A | 2 | $X_{A1}$ | $X_{A2}$ | 92,990 | 7,070 |
| B | 3 | $Y_B$ | $X_{B1}, X_{B2}$ | 27,570 | 13,240 |
| B | 4 | $X_{B1}$ | $X_{B2}$ | 87,020 | 34,650 |

Determine which one of the following statements best describes the data.

- A) Collinearity is present in both data sets A and B.
- B) Collinearity is present in neither data set A nor B.
- C) Collinearity is present in data set A only.
- D) Collinearity is present in data set B only.
- E) The degree of collinearity cannot be determined from the information given.

## Explanation

Collinearity is detected by regressing one predictor on the other and examining the resulting $R^2$ (or, equivalently, the variance inflation factor $\text{VIF} = 1/(1 - R^2)$). The relevant regressions are Model 2 (data set A) and Model 4 (data set B):

**Data set A** (Model 2): $R^2 = 1 - \dfrac{7{,}070}{92{,}990} = 0.924$, so $\text{VIF} = \dfrac{1}{1-0.924} \approx 13.2$.

**Data set B** (Model 4): $R^2 = 1 - \dfrac{34{,}650}{87{,}020} = 0.602$, so $\text{VIF} = \dfrac{1}{1-0.602} \approx 2.5$.

A VIF above the usual threshold of 10 signals serious collinearity. Data set A's VIF of $13.2$ indicates collinearity, while data set B's VIF of $2.5$ does not. Collinearity is present in **data set A only**, so the answer is **C**.
