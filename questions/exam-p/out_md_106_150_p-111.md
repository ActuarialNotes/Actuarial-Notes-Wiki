Let $X$ be a continuous random variable with density function
$$f(x) = \begin{cases} \frac{p-1}{x^p}, & x > 1 \\ 0, & \text{otherwise.} \end{cases}$$

Calculate the value of $p$ such that $E[X] = 2$.

- A) 1
- B) 2.5
- C) 3
- D) 5
- E) There is no such p.

## Explanation

The expected value is given by the integral of $x f(x)$:
$$E[X] = \int_{1}^{\infty} x \frac{p-1}{x^p} dx = (p-1) \int_{1}^{\infty} x^{1-p} dx$$

For the expected value to be finite, we must have $1-p < -1$, meaning $p > 2$.
$$E[X] = (p-1) \left[ \frac{x^{2-p}}{2-p} \right]_1^{\infty}$$
Since $p > 2$, $x^{2-p} \to 0$ as $x \to \infty$.
$$E[X] = (p-1) \left( 0 - \frac{1^{2-p}}{2-p} \right) = \frac{p-1}{p-2}$$

We are given $E[X] = 2$:
$$\frac{p-1}{p-2} = 2 \implies p - 1 = 2(p - 2) \implies p - 1 = 2p - 4 \implies p = 3$$
This satisfies $p > 2$, so $p = 3$ is the valid solution.
