# Historical OSFI Data

On June 10, 2026, **OSFI** discontinued its **Beyond 20/20** web server, which hosted historical financial data for federally regulated **Life Insurance** and **Property and Casualty Insurance** companies, covering 2020Q1 through 2025Q3:

https://osfi.beyond2020.com/

## Retirement Notice

Message from the website before it shut down:

> **Detailed Historical OSFI Data**
>
> Welcome to the source for detailed financial information, originating from the Office of the Superintendent of Financial Institutions (OSFI), filed by federally regulated **Life Insurance** and **Property and Casualty Insurance** companies. The information covers federally regulated Canadian insurance companies as well as foreign insurance companies operating within Canada. The information is "as filed" by the insurer.
>
> Effective June 10, 2026, this website will be retired. Please use www.osfi-bsif.gc.ca for all future requests.

## Data Format

> The data files available on this site are in ASCII format (text files) and are available from 2000 to the present.
>
> Interim data is available for Q1, Q2 and Q3 and is a subset of the data collected for the Annual Return (Q4).
>
> The data is organized into folders where each folder contains a complete information package for a specific period. The information package is composed of a record layout file, data file, a code and label file for the company names, and a blank return (.xls) for the data file. A page, row and column number called a data point address identifies each data item of a blank return. The company code in conjunction with a data point address uniquely identifies each data element.
>
> To save download time, a single .zip file is also included in the folder and contains all the individual files from the folder.

## Accessing the Data

> There is no charge for this data but you must self register to gain access to the site.
>
> 1. Login to the site. If it is your first visit to the site, select the Register link to create an account.
> 2. Once logged in, navigate through the folder structure until you find the data you want.
> 3. Click on the .zip file or its components to download the data.
