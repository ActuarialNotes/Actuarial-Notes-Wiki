Three separate GLMs are fit using the following model form:

$$g(Y) = \beta_0 + \beta_1 X_1 + \beta_2 X_2$$

The following error distributions were used for the three GLMs. Each model also used their canonical link functions:

- Model I: gamma
- Model II: Poisson
- Model III: binomial

When fit to the data, all three models resulted in the same parameter estimates:

| | |
|---|---|
| $\hat\beta_0$ | 2.0 |
| $\hat\beta_1$ | 1.0 |
| $\hat\beta_2$ | -1.0 |

Determine the correct ordering of the models' predicted values at observed point $(X_1, X_2) = (2, 1)$.

- A) I < II < III
- B) I < III < II
- C) II < I < III
- D) II < III < I
- E) The answer is not given by (A), (B), (C), or (D)

## Explanation

The linear predictor is the same for all three models:

$$\eta = \beta_0 + \beta_1(2) + \beta_2(1) = 2 + 2 - 1 = 3.$$

Each model inverts its **canonical link** to obtain the predicted mean $\hat\mu$:

- **Model I (gamma)** — canonical link $g(\mu) = 1/\mu$: $\hat\mu_I = 1/\eta = 1/3 \approx 0.333$.
- **Model II (Poisson)** — canonical link $g(\mu) = \ln\mu$: $\hat\mu_{II} = e^{\eta} = e^3 \approx 20.09$.
- **Model III (binomial)** — canonical link logit: $\hat\mu_{III} = \dfrac{e^{\eta}}{1+e^{\eta}} = \dfrac{e^3}{1+e^3} \approx 0.953$.

Ordering: $\hat\mu_I (0.333) < \hat\mu_{III} (0.953) < \hat\mu_{II} (20.09)$, i.e. $\text{I} < \text{III} < \text{II}$. The answer is **B**.
