An investor will receive $1,000 in 3 years. Using an annual effective interest rate of 5%, what is the present value of this payment?

- A) 863.84
- B) 857.34
- C) 950.00
- D) 1,157.63

## Explanation

Using the present value formula with compound interest:

$$PV = \frac{FV}{(1+i)^n} = \frac{1000}{(1.05)^3} = \frac{1000}{1.157625} \approx 863.84$$
