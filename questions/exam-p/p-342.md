Let $X$ be a Poisson random variable with cumulative distribution function $F$ such that $F(2)/F(1) = 2.6$.

Calculate $E(X)$.

- A) 3.2
- B) 4.0
- C) 4.2
- D) 5.0
- E) 5.2

## Explanation

$F(1) = e^{-\lambda}(1+\lambda)$ and $F(2) = e^{-\lambda}(1+\lambda+\lambda^2/2)$.

$$\frac{F(2)}{F(1)} = \frac{1+\lambda+\lambda^2/2}{1+\lambda} = 2.6$$

$$1+\lambda+\frac{\lambda^2}{2} = 2.6(1+\lambda) = 2.6+2.6\lambda$$

$$\frac{\lambda^2}{2}-1.6\lambda-1.6 = 0 \implies \lambda^2-3.2\lambda-3.2 = 0$$

$$\lambda = \frac{3.2+\sqrt{10.24+12.8}}{2} = \frac{3.2+4.8}{2} = \mathbf{4.0}$$

Verification: $F(2)/F(1) = 13e^{-4}/(5e^{-4}) = 13/5 = 2.6$ ✓
