You are estimating the coefficients of a linear regression model by minimizing the sum:

$$\sum_{i=1}^{n}\left(y_i - \beta_0 - \sum_{j=1}^{p}\beta_j x_{ij}\right)^2 \quad \text{subject to} \quad \sum_{j=1}^{p}\beta_j^2 \le s$$

From this model you have produced the following plot of various statistics as a function of the budget parameter, $s$, ranging from $s = 0$ to $s = 1\mathrm{e}{+}10$:

![Plot of two curves against budget parameter s: solid curve X decreases monotonically from about 1.15 to 0.3; dotted curve Y dips to a minimum near the middle before rising back toward 1.4](https://raw.githubusercontent.com/ActuarialNotes/Actuarial-Notes-Wiki/main/Media/Attachments/masi-2018s-q34-ridge-curves.png)

Determine which of the following statistics $X$ and $Y$ represent.

- A) X = Squared Bias, Y = Test MSE
- B) X = Test MSE, Y = Variance
- C) X = Training MSE, Y = Squared Bias
- D) X = Training MSE, Y = Variance
- E) X = Variance, Y = Training MSE

## Explanation

This is **ridge regression**: the constraint $\sum \beta_j^2 \le s$ relaxes as the budget $s$ increases, so larger $s$ means a more flexible (less regularized) model.

As flexibility increases:
- **Squared bias** decreases monotonically (most biased when heavily constrained near $s = 0$).
- **Variance** increases monotonically.
- **Test MSE** $=$ bias$^2$ $+$ variance $+$ irreducible error, which is **U-shaped** (decreases then increases).

Curve $X$ decreases monotonically, matching **squared bias**. Curve $Y$ is U-shaped with a minimum, matching **test MSE**. The answer is **A**.
