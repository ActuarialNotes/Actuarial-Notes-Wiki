An actuary tests H₀: 'current rates are adequate.' The rates truly ARE adequate, but the test rejects H₀ and the company raises rates. What error is this?

- A) Rejecting a null hypothesis that is actually true (a false positive)
- B) Failing to reject a null hypothesis that is actually false (a false negative)
- C) Choosing too small a significance level α
- D) Using the wrong test statistic for the data

<!-- rationale: 1: that is a Type II error (fail to reject a false H₀) · 2: a design choice, not an error that occurred · 3: an unrelated modeling mistake -->
