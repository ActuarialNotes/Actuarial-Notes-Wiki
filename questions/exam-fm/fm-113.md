Which of the following is an expression for the present value of a perpetuity with annual payments of 1, 2, 3, …, where the first payment will be made at the end of $n$ years, using an annual effective interest rate of $i$?

- A) $\dfrac{\ddot{a}_{\overline{n}|} - nv^n}{i}$
- B) $\dfrac{n - a_{\overline{n}|}}{i}$
- C) $\dfrac{v^n}{d}$
- D) $\dfrac{v^n}{d^2}$
- E) $\dfrac{v^n}{di}$

## Explanation

The present value is:
$$v^{n-1}/i + v^n/i + v^{n+1}/i + \cdots = (v^{n-2}/i)(v + v^2 + v^3 + \cdots) = \frac{v^{n-2}}{i} \cdot \frac{v}{1-v} = \frac{v^{n-1}}{i} \cdot \frac{1}{1-v}$$

Since $d = 1 - v$ and $v = 1/(1+i)$:
$$= \frac{v^{n-2}}{i^2} = \frac{v^n}{(vi)^2} = \frac{v^n}{d^2}$$

because $vi = v \cdot i = \frac{i}{1+i} = d$.
