An actuary has done an analysis of all policies that cover two cars. 70% of the policies are of type A for both cars, and 30% are type B for both cars. Claims on different cars across all policies are mutually independent. The distributions of the number of claims on a car are:

| Claims | Type A | Type B |
|---|---|---|
| 0 | 40% | 25% |
| 1 | 30% | 25% |
| 2 | 20% | 25% |
| 3 | 10% | 25% |

Four policies are selected at random. Calculate the probability that exactly one of the four policies has the same number of claims on both covered cars.

- A) 0.104
- B) 0.250
- C) 0.285
- D) 0.417
- E) 0.739

## Explanation

For a type A policy: $P(\text{same claims}) = 0.40^2 + 0.30^2 + 0.20^2 + 0.10^2 = 0.30$.

For a type B policy: $P(\text{same claims}) = 4 \times 0.25^2 = 0.25$.

For a random policy: $p = 0.70(0.30) + 0.30(0.25) = 0.21 + 0.075 = 0.285$.

For exactly 1 of 4 policies having same-count claims ($N \sim \text{Binomial}(4, 0.285)$):
$$P(N=1) = \binom{4}{1}(0.285)^1(0.715)^3 = 4(0.285)(0.3649) \approx \mathbf{0.417}$$
