The random variables $X$ and $Y$ have joint probability function $p(x,y)$ for $x = 0, 1$ and $y = 0, 1, 2$.

Suppose $3p(1,1) = p(1,2)$, and $p(1,1)$ maximizes the variance of $XY$.

Calculate the probability that $X$ or $Y$ is 0.

- A) $11/25$\n- B) $23/50$\n- C) $23/49$\n- D) $26/49$\n- E) $14/25$

## Explanation

Let $a = p(1,1)$, so $p(1,2) = 3a$. Then $p(1,0) + p(0,\cdot) = 1 - 4a$.

$XY$ takes values 0 (prob $1-4a$), 1 (prob $a$), 2 (prob $3a$).
$$E[XY] = a + 6a = 7a, \quad E[(XY)^2] = a + 12a = 13a$$
$$\text{Var}(XY) = 13a - 49a^2$$

Maximize: $\frac{d}{da}(13a - 49a^2) = 13 - 98a = 0 \implies a = \frac{13}{98}$

Check: $1 - 4a = 1 - 52/98 = 46/98 \geq 0$ ✓.

$$P(X = 0 \text{ or } Y = 0) = 1 - P(X=1, Y \geq 1) = 1 - 4a = 1 - \frac{52}{98} = \frac{46}{98} = \frac{23}{49}$$
