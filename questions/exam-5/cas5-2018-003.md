Given the following:

| Effective Date | Overall Average Rate Change | Rate Per Exposure ($) | Class Factor X | Class Factor Y | Class Factor Z | Expense Fee ($) |
|----------------|----------------------------|----------------------|----------------|----------------|----------------|-----------------|
| January 1, 2016 | -0.7% | 1,000 | 1.20 | 0.85 | 1.00 | 120 |
| July 1, 2017 | 10.0% | 1,112 | 1.20 | 0.85 | 1.00 | 120 |
| October 1, 2017 | — | 1,175 | 1.10 | 0.75 | 1.00 | 120 |
| April 1, 2018 | 1.0% | 1,175 | 1.10 | 0.75 | 1.00 | 132 |

- All policies are semi-annual.
- Exposures are written uniformly throughout the year.
- Expense fee is a per exposure fee that is added in the last step of the rate calculation.

## Part a (0.75 points)

Calculate the on-level factor for calendar year 2017 earned premium using the parallelogram method.

### Answer
1.0839

### Explanation
The parallelogram method weights each rate level by its proportion of the earned premium in the experience period, then computes the factor needed to bring the average historical level up to the current rate level.

**Identify rate levels relative to the most recent level:**

There are two rate changes affecting CY 2017 earned premium:
- July 1, 2017: overall +10.0% change, bringing the cumulative level to $1.10$ relative to the Jan 1, 2016 base.
- April 1, 2018: overall +1.0% change (after CY 2017), bringing the current cumulative level to $1.10 \times 1.01 = 1.111$.

The October 1, 2017 change altered class factors but not the overall average rate — it is addressed separately and is the subject of Part c.

**Compute weights for CY 2017 earned premium (semi-annual policies, uniform writings):**

For semi-annual policies written uniformly, the parallelogram for CY 2017 gives:
- At rate level 1.00 (pre-July 1, 2017): the triangle below the July 1, 2017 line within CY 2017 = $\frac{1}{2} \times \frac{6}{12} \times \frac{6}{12} + \frac{6}{12} \times \frac{6}{12} = 0.75$
  
  More simply: H1 2017 earns entirely at the old level (Jan–Jun written, earning Jan–Jun), and H2 2017 earns a 50/50 mix. For semi-annual policies written uniformly from Jan through Jun, all earn before July 1. For policies written in Q3 (Jul–Sep), the 10% rate change occurs at the start of their writing period. The result is that 75% of CY 2017 earned premium is at rate level 1.00 and 25% is at the +10% level.

| Rate Level (Relative) | Weight |
|----------------------|--------|
| 1.00 | 0.75 |
| 1.10 | 0.25 |
| 1.111 (current) | 0 |

**Weighted average historical rate level:**

$$\bar{L} = 0.75 \times 1.00 + 0.25 \times 1.10 + 0 \times 1.111 = 0.75 + 0.275 = 1.025$$

**On-level factor:**

$$\text{OLF} = \frac{\text{Current Level}}{\text{Average Historical Level}} = \frac{1.10 \times 1.01}{1.025} = \frac{1.111}{1.025} \approx 1.0839$$

### Examiner Report
Candidates were expected to identify each rate level at different points in time and weight them to calculate the on-level factor.

A common mistake was miscalculating weights (portion of year) to apply to each rate level.

## Part b (0.5 points)

Calculate the on-level factor for a policy effective on April 1, 2017 within Class Y using the extension of exposure method.

### Answer
1.0446

### Explanation
The extension of exposure method calculates an on-level factor for a specific policy by computing the ratio of the current-level premium to the premium at the level in effect when the policy was written.

**Premium at the time of writing (April 1, 2017):**

On April 1, 2017, the rates in effect are from the January 1, 2016 filing (the July 1, 2017 filing has not yet taken effect):
- Base rate per exposure: $1,000
- Class Y factor: $0.85$
- Expense fee: $120$

$$\text{Written Premium} = 1{,}000 \times 0.85 + 120 = 850 + 120 = 970$$

**Premium at the current rate level:**

Current rates (as of April 1, 2018, the most recent filing):
- Base rate per exposure: $1,175$
- Class Y factor: $0.75$ (changed October 1, 2017)
- Expense fee: $132$ (changed April 1, 2018)

$$\text{Current Premium} = 1{,}175 \times 0.75 + 132 = 881.25 + 132 = 1{,}013.25$$

**On-level factor:**

$$\text{OLF} = \frac{1{,}013.25}{970} \approx 1.0446$$

### Examiner Report
Candidates were expected to calculate written premium on different effective dates for a class, then calculate the on-level factor.

Common mistakes include:
- Using wrong point in time (row in table) or class factors.
- Not including the expense fee.

## Part c (0.5 points)

Assess the appropriateness of using the parallelogram method to calculate indicated class factors using the loss ratio method.

### Explanation
The parallelogram method is **not appropriate** for calculating indicated class factors when rate changes differ by class.

The parallelogram method applies the overall average rate change to bring all historical earned premium to the current level. It does not account for rate changes that varied by class — specifically, the October 1, 2017 change altered the class factors for X and Y differently (X decreased from 1.20 to 1.10; Y decreased from 0.85 to 0.75; Z was unchanged at 1.00). Applying an overall average on-level factor to class-level data would produce incorrect results because the adjustment needed differs by class.

The parallelogram method is appropriate if applied separately at the class level using the specific rate impact for each class. Applied at the aggregate level using the overall average rate change, it cannot properly adjust for class-specific factor changes, making it inappropriate for the class-level loss ratio method.

### Examiner Report
Candidates were expected to understand that the parallelogram method is applied at the aggregate level using the average rate change.

Common mistakes include:
- Not identifying the different class factor changes.
- Not realizing the parallelogram method uses overall average rate change.
