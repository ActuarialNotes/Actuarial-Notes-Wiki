You fit a linear model using the following two-level categorical variables:

$$X_1 = \begin{cases} 1 & \text{if Account} \\ 0 & \text{if Monoline} \end{cases} \qquad X_2 = \begin{cases} 1 & \text{if Multi-Car} \\ 0 & \text{if Single Car} \end{cases}$$

with the equation:

$$E(Y) = \beta_0 + \beta_1 X_1 + \beta_2 X_2 + \beta_3 X_1 X_2$$

This model produced the following parameter estimates:

$$\hat\beta_0 = -0.10, \quad \hat\beta_1 = -0.25, \quad \hat\beta_2 = 0.58, \quad \hat\beta_3 = -0.20.$$

Another actuary modeled the same underlying data, but coded the variables differently as such:

$$Z_1 = \begin{cases} 0 & \text{if Account} \\ 1 & \text{if Monoline} \end{cases} \qquad Z_2 = \begin{cases} 0 & \text{if Multi-Car} \\ 1 & \text{if Single Car} \end{cases}$$

with the equation:

$$E(Y) = \alpha_0 + \alpha_1 Z_1 + \alpha_2 Z_2 + \alpha_3 Z_1 Z_2$$

Afterwards you made a comparison of the individual parameter estimates in the two models.

Calculate how many pairs of coefficient estimates $(\hat\alpha_i, \hat\beta_i)$ switched signs, and how many pairs of estimates stayed identically the same, when results of the two models are compared.

- A) 1 sign change, 0 identical estimates
- B) 1 sign change, 1 identical estimate
- C) 2 sign changes, 0 identical estimates
- D) 2 sign changes, 1 identical estimate
- E) The answer is not given by (A), (B), (C), or (D)

## Explanation

Both codings must reproduce the same four group means. Note $Z_1 = 1 - X_1$ and $Z_2 = 1 - X_2$, so the reference cell flips. Compute the group means from Model 1, then read off the $\alpha$'s.

Model 1 group means:

$$\text{Account, Multi: } \beta_0+\beta_1+\beta_2+\beta_3 = 0.03, \quad \text{Account, Single: } \beta_0+\beta_1 = -0.35,$$
$$\text{Monoline, Multi: } \beta_0+\beta_2 = 0.48, \quad \text{Monoline, Single: } \beta_0 = -0.10.$$

In Model 2, the baseline ($Z_1 = Z_2 = 0$) is Account, Multi:

$$\alpha_0 = 0.03,\quad \alpha_1 = 0.48 - 0.03 = 0.45,\quad \alpha_2 = -0.35 - 0.03 = -0.38,$$
$$\alpha_0 + \alpha_1 + \alpha_2 + \alpha_3 = -0.10 \implies \alpha_3 = -0.20.$$

Compare term by term:

| | $\hat\beta_i$ | $\hat\alpha_i$ | Sign change? |
|---|---|---|---|
| 0 | $-0.10$ | $+0.03$ | yes |
| 1 | $-0.25$ | $+0.45$ | yes |
| 2 | $+0.58$ | $-0.38$ | yes |
| 3 | $-0.20$ | $-0.20$ | no (identical) |

There are **3 sign changes and 1 identical estimate**, which is not among choices A–D. The answer is **E**.
