An investor pays 4000 today for a three-year investment that returns cash flows of 1400 at the end of each year. The cash flows can be reinvested at the positive annual effective interest rate of $i$. Using an annual effective rate of interest of 4%, the net present value of this investment is 0.

Calculate $i$.

- A) 2.5%
- B) 3.0%
- C) 3.5%
- D) 4.0%
- E) 7.0%

## Explanation

NPV $= 0$ at 4% means the investment yields exactly 4%: $4000 = 1400a_{\overline{3}|0.04}$ ✓ (since $a_{\overline{3}|0.04} = 2.7751$ and $1400 \times 2.7751 = 3885$... hmm).

Actually: $4000(1.04)^3 = 1400[1 + (1+i) + (1+i)^2]$

$$4000(1.124864) = 1400[1 + (1+i) + (1+i)^2]$$

$$1 + (1+i) + (1+i)^2 = 4499.46/1400 = 3.2139$$

Let $x = 1+i$: $x^2 + x + 1 = 3.2139$, so $x^2 + x - 2.2139 = 0$.

$$x = \frac{-1 + \sqrt{1+4(2.2139)}}{2} = \frac{-1+\sqrt{9.8556}}{2} = \frac{-1+3.1393}{2} = 1.0697$$

$$i = 6.97\% \approx 7.0\%$$
