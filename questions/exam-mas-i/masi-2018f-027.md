You are given the following three functions of a random variable, $y$, where $-\infty < y < \infty$.

- I. $g(y) = 2 + 3y + 3(y - 5)^2$
- II. $g(y) = 4 - 4y$
- III. $g(y) = |y|$

Determine which of the above could be used as link functions in a GLM.

- A) I only
- B) II only
- C) III only
- D) I, II and III
- E) The answer is not given by (A), (B), (C), or (D)

## Explanation

A link function must be **monotonic** (and differentiable) so that it is invertible — the mean must be uniquely recoverable from the linear predictor.

- **I.** $g(y) = 2 + 3y + 3(y-5)^2$ is quadratic, so it is not monotonic — invalid.
- **II.** $g(y) = 4 - 4y$ is linear (strictly decreasing), hence monotonic and invertible — valid.
- **III.** $g(y) = |y|$ is V-shaped, not monotonic — invalid.

Only II qualifies, so the answer is **B**.
