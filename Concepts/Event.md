An **event** is any subset $E$ of the [[Sample Space]] $S$. Events are the objects to which probabilities are assigned.

A **simple event** contains exactly one outcome. A **compound event** contains two or more outcomes. The [[Axioms of Probability]] define $P(E) \in [0,1]$ for every event $E$, with $P(S) = 1$.

> [!example]- Rolling a Die {💡 Example}
> A fair six-sided die is rolled. The sample space is $S = \{1,2,3,4,5,6\}$.
>
> > [!answer]- Answer
> > Let $E = \{2, 4, 6\}$ be the event "an even number is rolled." This is a compound event containing 3 outcomes.
> > $$P(E) = \frac{|E|}{|S|} = \frac{3}{6} = 0.5$$
