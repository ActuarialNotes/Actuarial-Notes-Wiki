A 3-out-of-50 system is placed in series with a 48-out-of-50-system.

Calculate the number of minimal path sets.

- A) Fewer than 20,000
- B) At least 20,000 but fewer than 100,000
- C) At least 200,000 but fewer than 2,000,000
- D) At least 2,000,000 but fewer than 20,000,000
- E) At least 20,000,000

## Explanation

For a $k$-out-of-$n$ system, a minimal path set is any set of exactly $k$ working components, so there are $\binom{n}{k}$ minimal path sets.

- 3-out-of-50: $\binom{50}{3} = 19{,}600$ minimal path sets.
- 48-out-of-50: $\binom{50}{48} = \binom{50}{2} = 1{,}225$ minimal path sets.

For two systems in **series**, both subsystems must function, so each minimal path set of the combined system pairs a minimal path set from each subsystem:

$$\binom{50}{3} \times \binom{50}{2} = 19{,}600 \times 1{,}225 = 24{,}010{,}000.$$

This is at least $20{,}000{,}000$, so the answer is **E**.
