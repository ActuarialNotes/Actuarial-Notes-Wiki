Given the following for an insurance company:

$$\text{Premium} = (\text{Base Rate}) \times (\text{Rating Factor 1}) \times (\text{Rating Factor 2})$$

**Variable 1 Rating Factors (current):**

| Variable 1 Segment | Rating Factor 1 |
|--------------------|-----------------|
| A | 0.90 |
| B | 2.00 |
| C | 1.00 |

**Variable 2 Rating Factors (current):**

| Variable 2 Segment | Rating Factor 2 |
|--------------------|-----------------|
| X | 0.75 |
| Y | 0.95 |
| Z | 1.00 |

**Earned Exposures:**

| Variable 2 \ Variable 1 | A | B | C |
|--------------------------|-----|-------|-----|
| X | 800 | 1,500 | 600 |
| Y | 300 | 750 | 500 |
| Z | 500 | 500 | 1,325 |

**Reported Loss & ALAE ($):**

| Variable 2 \ Variable 1 | A | B | C |
|--------------------------|---------|-----------|-----------|
| X | 320,000 | 1,535,000 | 400,000 |
| Y | 170,000 | 1,535,000 | 700,000 |
| Z | 305,000 | 1,665,000 | 1,400,000 |

- The base rate is $1,000.
- The base classification for Variable 1 is C.
- The proposed overall rate level change is 0%.

## Part a (1.5 points)

Calculate the indicated relativities for Variable 1 using the adjusted pure premium method.

### Answer

| Variable 1 Segment | Adjusted Exposures | Reported Loss/ALAE | Adjusted Pure Premium | Indicated Relativity |
|--------------------|-------------------|-------------------|----------------------|----------------------|
| A | 1,385 | 795,000 | 574.01 | 0.5568 |
| B | 2,338 | 4,735,000 | 2,025.67 | 1.9649 |
| C (base) | 2,425 | 2,500,000 | 1,030.93 | 1.0000 |
| **Total** | **6,148** | **8,030,000** | **1,306.22** | |

### Explanation

**Step 1: Calculate adjusted exposures for each Variable 1 segment.**

Adjusted exposures weight each cell's earned exposures by the current Variable 2 rating factor, to remove the distorting effect of Variable 2's mix of business within each Variable 1 segment.

$$\text{Adjusted Exposure}_{A} = 800 \times 0.75 + 300 \times 0.95 + 500 \times 1.00 = 600 + 285 + 500 = 1{,}385$$

$$\text{Adjusted Exposure}_{B} = 1{,}500 \times 0.75 + 750 \times 0.95 + 500 \times 1.00 = 1{,}125 + 712.5 + 500 = 2{,}337.5 \approx 2{,}338$$

$$\text{Adjusted Exposure}_{C} = 600 \times 0.75 + 500 \times 0.95 + 1{,}325 \times 1.00 = 450 + 475 + 1{,}500 = 2{,}425$$

**Step 2: Compute total reported Loss/ALAE by Variable 1 segment.**

$$\text{Loss/ALAE}_{A} = 320{,}000 + 170{,}000 + 305{,}000 = 795{,}000$$

$$\text{Loss/ALAE}_{B} = 1{,}535{,}000 + 1{,}535{,}000 + 1{,}665{,}000 = 4{,}735{,}000$$

$$\text{Loss/ALAE}_{C} = 400{,}000 + 700{,}000 + 1{,}400{,}000 = 2{,}500{,}000$$

**Step 3: Calculate adjusted pure premiums.**

$$\text{APP}_{A} = \frac{795{,}000}{1{,}385} = 574.01$$

$$\text{APP}_{B} = \frac{4{,}735{,}000}{2{,}338} = 2{,}025.67$$

$$\text{APP}_{C} = \frac{2{,}500{,}000}{2{,}425} = 1{,}030.93$$

$$\text{APP}_{\text{Total}} = \frac{8{,}030{,}000}{6{,}148} = 1{,}306.22$$

**Step 4: Calculate indicated relativities to overall, then rebase to the base class C.**

Indicated relativity to overall:

$$r_{A} = \frac{574.01}{1{,}306.22} = 0.4394, \quad r_{B} = \frac{2{,}025.67}{1{,}306.22} = 1.5508, \quad r_{C} = \frac{1{,}030.93}{1{,}306.22} = 0.7892$$

Rebase to class C (divide each by $r_C = 0.7892$):

$$\text{Indicated Relativity}_{A} = \frac{0.4394}{0.7892} = 0.5568$$

$$\text{Indicated Relativity}_{B} = \frac{1.5508}{0.7892} = 1.9649$$

$$\text{Indicated Relativity}_{C} = \frac{0.7892}{0.7892} = 1.0000$$

### Examiner Report

Candidates were expected to understand the adjusted pure premium methodology and how to calculate class relativities.

Common mistakes include:
- Using unadjusted exposures (failing to weight by the Variable 2 rating factors)
- Not using the base class to rebase indicated relativities (leaving them expressed relative to the overall average rather than to class C)

## Part b (0.5 points)

Calculate the proposed base rate assuming the company selects half of the indicated relativity change for each segment of Variable 1.

### Answer

$1,035

### Explanation

**Step 1: Calculate selected relativities (halfway between current and indicated).**

$$\text{Selected}_{A} = 0.90 + 0.5 \times (0.5568 - 0.90) = 0.90 - 0.1716 = 0.7284 \approx 0.73$$

$$\text{Selected}_{B} = 2.00 + 0.5 \times (1.9649 - 2.00) = 2.00 - 0.0176 = 1.9824 \approx 1.98$$

$$\text{Selected}_{C} = 1.00 + 0.5 \times (1.00 - 1.00) = 1.00$$

**Step 2: Calculate the current and selected weighted average relativities using adjusted exposures as weights.**

$$\text{Current weighted relativity} = \frac{1{,}385 \times 0.90 + 2{,}338 \times 2.00 + 2{,}425 \times 1.00}{6{,}148} = \frac{1{,}246.5 + 4{,}676.0 + 2{,}425.0}{6{,}148} = \frac{8{,}347.5}{6{,}148} \approx 1.358$$

$$\text{Selected weighted relativity} = \frac{1{,}385 \times 0.73 + 2{,}338 \times 1.98 + 2{,}425 \times 1.00}{6{,}148} = \frac{1{,}011.1 + 4{,}629.2 + 2{,}425.0}{6{,}148} = \frac{8{,}065.3}{6{,}148} \approx 1.312$$

**Step 3: Calculate the off-balance factor (OBF) and new base rate.**

With a 0% overall rate level change target, the base rate must be adjusted upward to offset the reduction in the selected weighted relativity:

$$\text{Off-balance factor} = \frac{\text{Current weighted relativity}}{\text{Selected weighted relativity}} = \frac{1.358}{1.312} \approx 1.035$$

$$\text{New base rate} = \$1{,}000 \times 1.035 = \$1{,}035$$

### Examiner Report

Candidates were expected to understand how to calculate a proposed base rate given a premium change goal and a constraint on the change in class relativities.

Common mistakes include:
- Using unadjusted (earned) exposures rather than adjusted exposures as weights for the total relativity calculation
- Using indicated relativities rather than selected relativities when computing the selected weighted average

## Part c (0.5 points)

Briefly describe two social criteria for evaluating the appropriateness of rating variables.

### Explanation

Any two of the following social criteria:

1. **Affordability:** Insurance should be affordable for all policyholders. A rating variable that results in unaffordably high premiums for a segment of the population — even if actuarially justified — may be socially unacceptable. Regulators may restrict variables that price certain groups out of the market.

2. **Controllability:** Policyholders should be able to influence the rating variable through their own behavior in order to obtain a lower rate. Variables that reflect fixed personal characteristics beyond a policyholder's control (e.g., age, gender, place of birth) are viewed as less socially appropriate than those reflecting choices the insured can make.

3. **Privacy:** The rating variable should not require collection of personal information that invades a policyholder's privacy. Variables that require sensitive personal data (medical history, financial records, etc.) may be socially objectionable even if predictive.

4. **Causality / Logical relationship:** There should be a logical or intuitive causal relationship between the rating variable and the expected losses. Variables with a clear causal link are easier to justify socially and to policyholders, whereas purely statistical correlations without a logical explanation may face social or regulatory resistance.

### Examiner Report

Candidates were expected to understand social criteria of rating variables.

Common mistakes include:
- Using legal criteria (e.g., whether the variable is permitted by statute) instead of social criteria
- Using operational criteria (e.g., ease of verification, cost of data collection) instead of social criteria
