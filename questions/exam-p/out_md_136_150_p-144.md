An insurer's medical reimbursements have density function $f$, where $f(x)$ is proportional to
$$x e^{-x^2}, \quad \text{for } 0 < x < 1$$
and 0 otherwise.

Calculate the mode of the medical reimbursements.

- A) 0.00
- B) 0.50
- C) 0.71
- D) 0.84
- E) 1.00

## Explanation

Let $f(x) = c x e^{-x^2}$. To find the mode, set the derivative of the density function equal to zero.

Using the product rule:
$$f'(x) = c \left[ \frac{d}{dx}(x) \cdot e^{-x^2} + x \cdot \frac{d}{dx}(e^{-x^2}) \right]$$
$$= c \left[ 1 \cdot e^{-x^2} + x(-2x)e^{-x^2} \right]$$
$$= c e^{-x^2} (1 - 2x^2)$$

Set $f'(x) = 0$:
$$1 - 2x^2 = 0 \implies 2x^2 = 1 \implies x^2 = 0.5 \implies x = \frac{1}{\sqrt{2}} \approx 0.707$$

Since $f'(x)$ changes from positive to negative at this point, it corresponds to the maximum (the mode). The mode is 0.71.
