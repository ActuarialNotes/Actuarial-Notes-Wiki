Given the following data evaluated as of December 31, 2018:

**Cumulative Reported Claim Counts as of (months):**

| Accident Year | 12 | 24 | 36 | 48 |
|---------------|----|----|----|----|
| 2015 | 250 | 238 | 245 | 260 |
| 2016 | 275 | 270 | | |
| 2017 | 323 | 320 | | |
| 2018 | 375 | | | |

**Cumulative Reported Claims (\$000s) as of (months):**

| Accident Year | 12 | 24 | 36 | 48 |
|---------------|----|----|----|----|
| 2015 | 1,250 | 1,280 | 1,325 | 1,430 |
| 2016 | 1,365 | 1,395 | 1,450 | |
| 2017 | 1,825 | 1,875 | | |
| 2018 | 1,900 | | | |

**Reported Claim Count Age-to-Age Factors:**

| Accident Year | 12-24 | 24-36 | 36-48 |
|---------------|-------|-------|-------|
| 2015 | 0.952 | 1.029 | 1.061 |
| 2016 | 0.982 | | |
| 2017 | 0.991 | | |

- \$700,000 Accident year 2018 paid claims to date.
- There is no development after 48 months.

## Part a (2 points)

Calculate an unpaid claims estimate for accident year 2018 using a frequency-severity technique.

### Answer
1482

### Explanation
**Step 1: Develop claim counts to ultimate**

Calculate age-to-age factors:

| AY | 12-24 | 24-36 | 36-48 |
|----|-------|-------|-------|
| 2015 | 238/250 = 0.952 | 245/238 = 1.029 | 260/245 = 1.061 |
| 2016 | 270/275 = 0.982 | | |
| 2017 | 320/323 = 0.991 | | |
| Selected | 0.975 | 1.030 | 1.061 |

(Averages of available factors)

Cumulative age-to-ultimate factors for AY 2018 (at 12 months):

$$\text{12-to-Ult} = 0.975 \times 1.030 \times 1.061 \times 1.000 = 1.065$$

Ultimate claim count for AY 2018:

$$375 \times 1.065 = 399$$

**Step 2: Build average severity triangle**

Divide cumulative reported claims (\$000s) by cumulative reported claim counts:

| AY | 12 | 24 | 36 | 48 |
|----|----|----|----|-----|
| 2015 | 1,250/250 = 5,000 | 1,280/238 = 5,378 | 1,325/245 = 5,408 | 1,430/260 = 5,500 |
| 2016 | 1,365/275 = 4,964 | 1,395/270 = 5,167 | 1,450/? = 5,216 | |
| 2017 | 1,825/323 = 5,031 | 1,875/320 = 5,234 | | |
| 2018 | 1,900/375 = 5,067 | | | |

**Step 3: Develop severity to ultimate**

Age-to-age severity factors:

| AY | 12-24 | 24-36 | 36-48 |
|----|-------|-------|-------|
| 2015 | 5,378/5,000 = 1.076 | 5,408/5,378 = 1.006 | 5,500/5,408 = 1.017 |
| 2016 | 5,167/4,964 = 1.041 | 5,216/5,167 = 1.010 | |
| 2017 | 5,234/5,031 = 1.040 | | |
| Selected | 1.052 | 1.008 | 1.017 |

Cumulative severity age-to-ultimate for AY 2018 (at 12 months):

$$\text{12-to-Ult} = 1.052 \times 1.008 \times 1.017 = 1.078$$

Ultimate severity:

$$5{,}067 \times 1.078 = 5{,}462 \approx 5{,}463$$

**Step 4: Calculate unpaid claims estimate**

$$\text{Ultimate Claims} = \text{Ultimate Count} \times \text{Ultimate Severity} = 399 \times 5{,}463 / 1{,}000 \approx \$2{,}182{,}000$$

$$\text{Unpaid Claims} = 2{,}182{,}000 - 700{,}000 = \$1{,}482{,}000$$

### Examiner Report
Candidates were expected to develop claim counts to ultimate, create an average severity triangle and develop severity to ultimate, multiply claim counts by severity, and subtract paid claims to date.

Common mistakes included:

- Calculating an "average" claim count (e.g., averaging across accident years) rather than developing the latest diagonal to ultimate.
- Using a single diagonal or a single value to derive ultimate severity instead of building an age-to-age development framework.
- Subtracting reported losses instead of paid losses.
- Omitting the final step of calculating unpaid claims and stopping at ultimate claims.
- Performing a reported loss development technique instead of a frequency-severity technique.

## Part b (0.25 points)

Briefly describe a situation that would lead to the downward development in reported claim counts observed from 12 to 24 months.

### Explanation
Claims closed without payment (also called closed without pay or CWP). When a claim is initially reported and opened but is subsequently determined to be fraudulent, outside coverage, or otherwise not payable, it is closed without any payment. This causes the cumulative reported claim count to decrease as those previously counted claims are removed from the open inventory, resulting in a count factor less than 1.0.

### Examiner Report
Candidates were expected to identify a situation that would cause reported claim counts to decrease between 12 and 24 months.

Common mistakes included:

- Explanations that would cause a restatement to the entire triangle (e.g., a law change, court decision, or deductible change) — these would not explain a decrease in counts at one specific development age.
- Vague or incomplete explanations that did not identify a specific mechanism causing the count to fall.
