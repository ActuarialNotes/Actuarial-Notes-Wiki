The lifetime of a light bulb has density function, $f$, where $f(x)$ is proportional to
$$\frac{x^2}{1+x^3}$$
for $0 < x < 5$, and 0 otherwise.

Calculate the mode of this distribution.

- A) 0.00
- B) 0.79
- C) 1.26
- D) 4.42
- E) 5.00

## Explanation

Let $f(x) = c \frac{x^2}{1+x^3}$. The mode is the value of $x$ that maximizes the density function. We find the maximum by setting the derivative to zero.

Using the quotient rule:
$$f'(x) = c \frac{\frac{d}{dx}(x^2) (1+x^3) - x^2 \frac{d}{dx}(1+x^3)}{(1+x^3)^2}$$
$$= c \frac{2x(1+x^3) - x^2(3x^2)}{(1+x^3)^2}$$
$$= c \frac{2x + 2x^4 - 3x^4}{(1+x^3)^2}$$
$$= c \frac{2x - x^4}{(1+x^3)^2} = c \frac{x(2 - x^3)}{(1+x^3)^2}$$

Set $f'(x) = 0$:
$$x(2 - x^3) = 0$$
The roots are $x=0$ (boundary) and $2 - x^3 = 0 \implies x^3 = 2 \implies x = 2^{1/3} \approx 1.2599$.

Since $x=2^{1/3}$ is in the interval $(0, 5)$ and $f'(x)$ changes from positive to negative at this point, it is a local maximum. Thus, the mode is 1.26.
