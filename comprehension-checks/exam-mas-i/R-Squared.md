You add a completely irrelevant predictor to a linear regression. What happens to R², and what does that reveal about it?

- A) R² increases or stays equal; R² never penalizes added predictors, so it can't flag a useless one
- B) R² decreases, correctly flagging the useless predictor
- C) R² stays exactly the same regardless of predictors
- D) R² measures out-of-sample accuracy, so it drops
