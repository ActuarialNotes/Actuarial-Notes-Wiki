Comparing two GLMs, Model A has AIC = 246 and Model B has AIC = 250. Which is preferred, and what does AIC balance?

- A) Model A (lower AIC); AIC balances goodness of fit against the number of parameters
- B) Model B (higher AIC); a higher value means more variance explained
- C) Model A, but only if the two models are nested
- D) Cannot tell — AIC measures only fit, not complexity
