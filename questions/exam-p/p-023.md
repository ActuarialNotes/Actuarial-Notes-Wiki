The number of injury claims per month is modeled by a random variable $N$ with

$$P[N = n] = \frac{1}{(n+1)(n+2)}, \quad \text{for nonnegative integers } n.$$

Calculate the probability of at least one claim during a particular month, given that there have been at most four claims during that month.

- A) 1/3
- B) 2/5
- C) 1/2
- D) 3/5
- E) 5/6

## Explanation

Note $P[N=n] = \frac{1}{n+1} - \frac{1}{n+2}$, so $P[N \leq k] = 1 - \frac{1}{k+2}$.

$$P[N \geq 1 \mid N \leq 4] = \frac{P[1 \leq N \leq 4]}{P[N \leq 4]}$$

$$P[1 \leq N \leq 4] = \frac{1}{6} + \frac{1}{12} + \frac{1}{20} + \frac{1}{30} = \frac{10+5+3+2}{60} = \frac{20}{60}$$

$$P[N \leq 4] = \frac{1}{2} + \frac{1}{6} + \frac{1}{12} + \frac{1}{20} + \frac{1}{30} = \frac{30+10+5+3+2}{60} = \frac{50}{60}$$

$$P[N \geq 1 \mid N \leq 4] = \frac{20/60}{50/60} = \frac{20}{50} = \frac{2}{5}$$
