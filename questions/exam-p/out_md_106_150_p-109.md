Automobile claim amounts are modeled by a uniform distribution on the interval $[0, 10,000]$. Actuary A reports $X$, the claim amount divided by 1000. Actuary B reports $Y$, which is $X$ rounded to the nearest integer from 0 to 10.

Calculate the absolute value of the difference between the $4^{\text{th}}$ moment of $X$ and the $4^{\text{th}}$ moment of $Y$.

- A) 0
- B) 33
- C) 296
- D) 303
- E) 533

## Explanation

$X$ has a uniform distribution on the interval $[0, 10]$.
The $4^{\text{th}}$ moment of $X$ is:
$$E[X^4] = \int_{0}^{10} x^4 \cdot \frac{1}{10} dx = \frac{1}{10} \left[ \frac{x^5}{5} \right]_0^{10} = \frac{100,000}{50} = 2000$$

$Y$ is $X$ rounded to the nearest integer. $Y$ takes values $0, 1, 2, \dots, 10$.
* $P(Y=0) = P(0 \le X < 0.5) = 0.05$
* $P(Y=10) = P(9.5 \le X \le 10) = 0.05$
* $P(Y=k) = P(k-0.5 \le X < k+0.5) = 0.1$ for $k=1,\dots,9$.

The $4^{\text{th}}$ moment of $Y$ is:
$$E[Y^4] = 0^4(0.05) + \sum_{k=1}^{9} k^4(0.1) + 10^4(0.05)$$
$$= 0.1(1^4 + 2^4 + 3^4 + 4^4 + 5^4 + 6^4 + 7^4 + 8^4 + 9^4) + 500$$
$$= 0.1(1 + 16 + 81 + 256 + 625 + 1296 + 2401 + 4096 + 6561) + 500 = 0.1(15333) + 500 = 2033.3$$

The absolute value of the difference is $|2000 - 2033.3| = 33.3$.
