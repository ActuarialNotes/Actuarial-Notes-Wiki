# Set Function

## Definition

A ==set function== assigns a real number to each set in a collection.

$$ f: \mathcal{F} \to \mathbb{R} $$

> [!example]- <u>Example</u>
> Let $\mathcal{F} = \{\emptyset, \{1\}, \{2\}, \{1,2\}\}$ and define $f(A) = |A|$ (the cardinality function). Then:
> - $f(\emptyset) = 0$
> - $f(\{1\}) = 1$
> - $f(\{2\}) = 1$
> - $f(\{1,2\}) = 2$
>
> A probability measure $P$ is a special set function where $P: \mathcal{F} \to [0,1]$ and $P(S) = 1$.
