A discrete random variable $X$ has the following probability distribution: $P(X=1) = 0.2$, $P(X=2) = 0.3$, $P(X=3) = 0.4$, $P(X=4) = 0.1$. What is $E[X]$?

- A) 2.0
- B) 2.4
- C) 2.5
- D) 3.0

## Explanation

The expected value is $E[X] = \sum x \cdot P(X = x)$:

$$E[X] = 1(0.2) + 2(0.3) + 3(0.4) + 4(0.1) = 0.2 + 0.6 + 1.2 + 0.4 = 2.4$$
