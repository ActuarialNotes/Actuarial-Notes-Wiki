# OSFI PC-1 Return (2023)

## Regulation Overview
Effective Q1 2023 (alongside the IFRS 17 transition), OSFI decommissioned the legacy P&C Quarterly and Annual Supplement returns and replaced them with a new family of uniform P&C returns: the **P&C Core Financial Statement Return (PC-1)**, **P&C Supervisory Quarterly Return (PC-2)**, **P&C Supervisory Annual Supplement Return (PC-3)**, and **P&C Provincial Return (PC-PROV)**. Every federally regulated P&C insurer must file these on a quarterly and annual basis.

PC-1 captures the core financial statements (balance sheet, statement of income, insurance contract liability roll-forwards) under IFRS 17 presentation, and is the primary aggregation point OSFI and CCIR use to compile industry-level benchmarks — loss ratios, expense ratios, and reserve adequacy trends — across the federally regulated P&C market.

## Strategic Impact
- **Financial Reporting:** Establishes the IFRS 17-aligned reporting taxonomy that all federally regulated insurers must use, replacing the pre-2023 P&C Annual/Quarterly Supplement formats.
- **Benchmarking:** Industry-aggregate PC-1/PC-3 data underpins the loss-trend and capital benchmarks regulators (OSFI, AIRB, GISA) reference when reviewing rate filings.
- **Actuarial:** Insurance contract liability roll-forwards on PC-1 feed directly into the Appointed Actuary's valuation work and the Statement of Actuarial Opinion.
