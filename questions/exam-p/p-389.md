The joint probability density function of $X$ and $Y$ is $f(x,y) = 8xy$ for $0 < y < x < 1$.

Calculate the marginal density of $X$ and find $E[Y \mid X = x]$.

- A) $f_X(x) = 4x^3$; $E[Y|X=x] = x/3$\n- B) $f_X(x) = 4x^3$; $E[Y|X=x] = 2x/3$\n- C) $f_X(x) = 2x^2$; $E[Y|X=x] = x/2$\n- D) $f_X(x) = 4x^3$; $E[Y|X=x] = x/2$\n- E) $f_X(x) = 2x^3$; $E[Y|X=x] = 2x/3$

## Explanation

$$f_X(x) = \int_0^x 8xy\,dy = 8x\cdot\frac{x^2}{2} = 4x^3$$

$$f_{Y|X}(y|x) = \frac{8xy}{4x^3} = \frac{2y}{x^2}$$

$$E[Y|X=x] = \int_0^x y\cdot\frac{2y}{x^2}\,dy = \frac{2}{x^2}\cdot\frac{x^3}{3} = \frac{2x}{3}$$
