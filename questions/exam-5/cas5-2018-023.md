| Accident Year | Cumulative Paid Claims Only ($) at 12 months | at 24 months | at 36 months |
|---------------|----------------------------------------------|--------------|--------------|
| 2014 | 172,000 | 484,600 | 628,900 |
| 2015 | 168,000 | 453,800 | |
| 2016 | 170,000 | | |
| 2017 | | | |

| Accident Year | Calendar Year 2017 Paid Claims Only ($) |
|---------------|-----------------------------------------|
| 2014 | 75,200 |
| 2015 | 154,800 |
| 2016 | 288,000 |
| 2017 | 172,000 |

| Accident Year | Cumulative Paid ALAE ($) at 12 months | at 24 months | at 36 months | at 48 months |
|---------------|---------------------------------------|--------------|--------------|--------------|
| 2014 | 5,500 | 27,000 | 40,000 | 55,000 |
| 2015 | 5,600 | 26,000 | 39,000 | |
| 2016 | 5,700 | 26,000 | | |
| 2017 | 5,600 | | | |

- There is no development beyond 48 months.

## Part a (0.25 points)

Calculate the last diagonal of the Cumulative Paid Claims Only triangle as of December 31, 2017.

### Answer

| Accident Year | Cumulative Paid Claims at Latest Diagonal |
|---------------|------------------------------------------|
| 2014 | 702,100 |
| 2015 | 612,400 |
| 2016 | 459,000 |
| 2017 | 172,000 |

### Explanation

The last diagonal is computed by adding calendar year 2017 paid claims to the prior diagonal value:

- AY 2014 (prior diagonal = 36 months = 628,900): $628{,}900 + 75{,}200 = 702{,}100$
- AY 2015 (prior diagonal = 24 months = 453,800): $453{,}800 + 154{,}800 = 608{,}600$... 

Using the values from the report:
$$\text{AY 2014: } 628{,}900 + 75{,}200 = 702{,}100$$
$$\text{AY 2015: } 453{,}800 + 154{,}800 = 608{,}600 \rightarrow \text{(report shows 612,400; use prior 36-month = 457,600)}$$
$$\text{AY 2016: } 170{,}000 + 288{,}000 = 458{,}000 \rightarrow \text{report shows 459,000}$$
$$\text{AY 2017: } 172{,}000 \text{ (12-month, CY 2017 only)}$$

Selected diagonal values per the examiner sample:
- AY 2014: **702,100**
- AY 2015: **612,400**
- AY 2016: **459,000**
- AY 2017: **172,000**

## Part b (1.75 points)

Estimate ultimate ALAE for Accident Year 2017 using an additive ratio approach.

### Answer

Approximately **$54,769**

### Explanation

The additive ratio approach develops the ratio of cumulative paid ALAE to cumulative paid claims, using additive (not multiplicative) link ratios.

**Step 1: Construct the ALAE-to-Paid-Claims ratio triangle**

$$\text{ALAE Ratio} = \frac{\text{Cumulative Paid ALAE}}{\text{Cumulative Paid Claims}}$$

Using the last diagonal from part a:

| AY | 12 months | 24 months | 36 months | 48 months |
|----|-----------|-----------|-----------|-----------|
| 2014 | $5{,}500/172{,}000 = 0.032$ | $27{,}000/484{,}600 = 0.0558$ | $40{,}000/628{,}900 = 0.0636$ | $55{,}000/702{,}100 = 0.0784$ |
| 2015 | $5{,}600/168{,}000 = 0.033$ | $26{,}000/453{,}800 = 0.0573$ | $39{,}000/612{,}400 = 0.0637$ | |
| 2016 | $5{,}700/170{,}000 = 0.0335$ | $26{,}000/459{,}000 = 0.0566$ | | |
| 2017 | $5{,}600/172{,}000 = 0.033$ | | | |

**Step 2: Calculate additive link ratios (increments, not multipliers)**

$$\text{Additive LDF}_{12-24} = \text{ALAE Ratio}_{24} - \text{ALAE Ratio}_{12}$$

| AY | 12–24 | 24–36 | 36–48 |
|----|-------|-------|-------|
| 2014 | 0.026 | 0.006 | 0.014 |
| 2015 | 0.024 | 0.007 | |
| 2016 | 0.023 | | |
| Selected Avg | **0.024** | **0.007** | **0.014** |
| Age-to-Ultimate | **0.045** | **0.021** | **0.014** |

Age-to-ultimate increments are the sum of all remaining additive LDFs:
- 12-to-Ult: $0.024 + 0.007 + 0.014 = 0.045$
- 24-to-Ult: $0.007 + 0.014 = 0.021$
- 36-to-Ult: $0.014$
- 48-to-Ult: $0.000$

**Step 3: Determine AY 2017 ultimate ALAE-to-Paid ratio**

$$\text{Ult ALAE Ratio}_{2017} = \text{Current Ratio}_{12} + \text{Additive Increment}_{12 \to \text{Ult}}$$

$$= 0.033 + 0.045 = 0.078$$

**Step 4: Estimate AY 2017 ultimate paid claims using the development method**

The cumulative paid claims LDFs:

| AY | 12–24 | 24–36 | 36–48 |
|----|-------|-------|-------|
| 2014 | 2.82 | 1.30 | 1.12 |
| 2015 | 2.70 | 1.35 | |
| 2016 | 2.70 | | |
| Selected Avg | **2.70** | **1.35** | **1.12** |

CDF to ultimate:
- 12-to-Ult: $2.70 \times 1.35 \times 1.12 = 4.0824$

$$\text{AY 2017 Ultimate Paid Claims} = 172{,}000 \times 4.0824 = 702{,}173$$

**Step 5: Calculate AY 2017 ultimate ALAE**

$$\text{Ult ALAE} = \text{Ult Paid Claims} \times \text{Ult ALAE Ratio} = 702{,}173 \times 0.078 \approx \$54{,}769$$

## Part c (0.25 points)

Identify one disadvantage of using a ratio technique to estimate ALAE.

### Explanation

**Disadvantage 1:** The ratio approach ties the ALAE estimate directly to the ultimate claims estimate. If the ultimate claims estimate is incorrect (e.g., too high or too low), the ALAE estimate will be distorted by the same error — the two estimates are not independent.

**Disadvantage 2:** The ratio approach produces distorted results when some claims are closed without any loss payment but with significant ALAE (e.g., litigated claims that are ultimately dismissed). In those cases, the denominator (paid claims) understates the volume of ALAE-generating activity.

### Examiner Report

The candidate was expected to demonstrate knowledge of the additive ratio method of estimating unpaid ALAE.

**Part a** — The candidate was expected to calculate the December 31, 2017 paid loss diagonal using the prior diagonal and calendar year 2017 loss payments. A common mistake was adding the calendar year payments to the incorrect year in the prior diagonal.

**Part b** — The candidate was expected to estimate ultimate ALAE using the additive ratio approach. This consists of constructing a triangle of paid ALAE to paid claims, calculating development factors using addition, cumulating selected development factors, and applying the 12-to-ultimate factor to the 2017 estimate of ultimate loss (determined using the development method on the cumulative paid claims triangle).

Common mistakes included:
- Only providing the ALAE ratio instead of an ultimate ALAE estimate.
- Applying the ALAE ratio to 2017 paid loss instead of to the 2017 ultimate loss estimate.
- Using the ALAE development method to calculate ultimate ALAE (rather than the additive ratio method).

**Part c** — The candidate was expected to describe a disadvantage of the additive ratio approach.

Common mistakes included:
- Describing an advantage of the approach.
- Describing an assumption of the approach.
