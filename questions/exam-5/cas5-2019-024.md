Given the following information:

**Cumulative Paid Claims Only as of (months) (\$000s):**

| Accident Year | 12    | 24     | 36     | 48     |
|---------------|-------|--------|--------|--------|
| 2015          | 3,556 | 15,960 | 17,150 | 17,556 |
| 2016          | 3,700 | 10,920 | 15,600 |        |
| 2017          | 3,430 | 11,808 |        |        |
| 2018          | 4,050 |        |        |        |

**Cumulative Paid ALAE as of (months) (\$000s):**

| Accident Year | 12  | 24  | 36  | 48  |
|---------------|-----|-----|-----|-----|
| 2015          | 355 | 520 | 562 | 569 |
| 2016          | 370 | 550 | 589 |     |
| 2017          | 343 | 496 |     |     |
| 2018          | 374 |     |     |     |

**Selected Ultimate Claims Only (\$000s):**

| Accident Year | Selected Ultimate |
|---------------|-------------------|
| 2015          | 17,500            |
| 2016          | 17,800            |
| 2017          | 17,600            |
| 2018          | 18,500            |

- There is no development beyond 48 months.

## Part a (1.25 points)

Estimate ultimate ALAE for accident year 2018 using the multiplicative ratio development approach.

### Answer
595.7

### Explanation

**Step 1: Calculate ALAE Ratios** (Cumulative Paid ALAE ÷ Selected Ultimate Claims)

$$\text{ALAE Ratio} = \frac{\text{Cumulative Paid ALAE}}{\text{Selected Ultimate Claims}}$$

| AY   | 12                        | 24                        | 36                        | 48                        |
|------|---------------------------|---------------------------|---------------------------|---------------------------|
| 2015 | $355/17{,}500 = 0.0203$   | $520/17{,}500 = 0.0297$   | $562/17{,}500 = 0.0321$   | $569/17{,}500 = 0.0325$   |
| 2016 | $370/17{,}800 = 0.0208$   | $550/17{,}800 = 0.0309$   | $589/17{,}800 = 0.0331$   |                           |
| 2017 | $343/17{,}600 = 0.0195$   | $496/17{,}600 = 0.0282$   |                           |                           |
| 2018 | $374/18{,}500 = 0.0202$   |                           |                           |                           |

**ALAE Ratio Triangle:**

| AY   | 12     | 24     | 36     | 48     |
|------|--------|--------|--------|--------|
| 2015 | 0.0203 | 0.0297 | 0.0321 | 0.0325 |
| 2016 | 0.0208 | 0.0309 | 0.0331 |        |
| 2017 | 0.0195 | 0.0282 |        |        |
| 2018 | 0.0202 |        |        |        |

**Step 2: Calculate Link Ratios (ratio age-to-age factors)**

$$f_{12-24}: \quad \frac{0.0297}{0.0203} = 1.463, \quad \frac{0.0309}{0.0208} = 1.486, \quad \frac{0.0282}{0.0195} = 1.446$$

$$\text{Selected (Avg) } f_{12-24} = \frac{1.463 + 1.486 + 1.446}{3} = 1.465$$

$$f_{24-36}: \quad \frac{0.0321}{0.0297} = 1.081, \quad \frac{0.0331}{0.0309} = 1.071$$

$$\text{Selected (Avg) } f_{24-36} = \frac{1.081 + 1.071}{2} = 1.076$$

$$f_{36-48}: \quad \frac{0.0325}{0.0321} = 1.012$$

$$\text{Selected } f_{36-48} = 1.012$$

Since there is no development beyond 48 months: $f_{48-\text{Ult}} = 1.000$

**Link Ratio Summary:**

| Age          | 12-24 | 24-36 | 36-48 |
|--------------|-------|-------|-------|
| AY 2015      | 1.463 | 1.081 | 1.012 |
| AY 2016      | 1.486 | 1.071 |       |
| AY 2017      | 1.446 |       |       |
| Selected     | 1.465 | 1.076 | 1.012 |

**Cumulative (12-to-Ultimate) factor:**

$$F_{12} = 1.465 \times 1.076 \times 1.012 = 1.595$$

**Step 3: Project AY 2018 Ultimate ALAE Ratio**

$$\text{2018 Ult. ALAE Ratio} = 0.0202 \times 1.595 = 0.0322$$

**Step 4: Calculate AY 2018 Ultimate ALAE**

$$\text{2018 Ult. ALAE} = 0.0322 \times 18{,}500 = 595.7 \text{ (\$000s)}$$

- Recalculating the ultimate claims instead of using the provided selected ultimate claims figure.

## Part b (0.25 points)

Briefly describe when it is more appropriate to use an additive development approach than a multiplicative development approach.

### Explanation
The **additive approach** is preferable when the ALAE ratios are very small, because a multiplicative approach can produce highly leveraged cumulative development factors (CDFs) in those situations. When early-maturity link ratios from the multiplicative approach are large (highly leveraged), the additive method will produce less leveraged factors and more stable projections at early maturities.

In other words: when the ratios at early ages are low (near zero), small absolute changes in the ratio translate to very large multiplicative link ratios. The additive method avoids this problem by working in absolute ratio units rather than multiplicative ones.

### Examiner Report
Candidates were expected to describe a situation when the additive method is preferable over the multiplicative method.

A common mistake was providing an advantage of the additive method without describing the specific situation in which it is preferable. For example, simply stating "more stable" or "less volatile" without explaining when and why.

## Part c (0.5 points)

Briefly describe one advantage and one disadvantage of the ratio techniques.

### Explanation
**Advantage** (any one of the following):

- **Recognizes the inherent relationship between losses and ALAE** — by expressing ALAE as a ratio to ultimate claims, the method captures the natural co-movement between the two, and the ratio is often more stable than raw ALAE dollar amounts.
- **Provides diagnostics**: if the ratio for any accident year appears unreasonable, it can be identified and excluded from the selected factors.
- **Less leveraged development factors** — the ratio development factors tend to be less highly leveraged than development factors based on paid ALAE dollars directly, reducing sensitivity to early-period volatility.
- **More stable/less volatile** than developing paid ALAE dollars directly, particularly for lines where ALAE payments at early maturities are erratic.

**Disadvantage** (any one of the following):

- **Error in ultimate claims propagates to ALAE** — an error in the selected ultimate claims estimate will introduce a corresponding error in the ultimate ALAE calculation.
- **Distorted ratios at early maturities**: if there are many ALAE payments at early maturity with no accompanying paid claims at that same maturity, the ratio will be artificially inflated or distorted.
- **Not accurate if ALAE has a different trend rate than claims costs** — the method assumes ALAE is proportional to ultimate claims, which fails when ALAE trends differ from loss trends.

### Examiner Report
Candidates were expected to provide one advantage and one disadvantage of the ratio method.

Common mistakes included:
- Providing advantages or disadvantages too generic to be meaningful — answers that would apply too broadly to many concepts or methods, such as "method is easier/harder to calculate," "data is hard to obtain," or "easy to explain to management."
- Noting that ALAE is not proportional to claims without referring to the volatility of the ratio and its effect on the selection of factors.
