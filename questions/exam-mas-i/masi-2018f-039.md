Two actuaries were given a dataset and asked to build a model to predict claim frequency using any of 5 independent predictors $\{1, 2, 3, 4, 5\}$ as well as an intercept $\{I\}$.

- Actuary A chooses their model using Best Subset Selection
- Actuary B chooses their model using Forward Stepwise Regression
- When evaluating the models they both used R-squared to compare models with the same number of parameters, and AIC to compare models with different numbers of parameters.

For each candidate model, the $R^2$ and log-likelihood are provided. The relevant best-per-size models are: size 1 — $(I, 2)$ with $R^2 = 0.57$, LL $= 1.4$; size 2 — $(I, 1, 5)$ with $R^2 = 0.69$, LL $= 3.0$, and $(I, 2, 5)$ with $R^2 = 0.68$, LL $= 2.9$; size 3 — $(I, 1, 4, 5)$ with $R^2 = 0.79$, LL $= 3.9$, and $(I, 2, 4, 5)$ with $R^2 = 0.75$, LL $= 3.45$; and so on up to the full model.

$\text{AIC}_j$ is the AIC of the model chosen by Actuary $j$.

Calculate the absolute value of the difference between $\text{AIC}_A$ and $\text{AIC}_B$.

- A) Less than 0.15
- B) At least 0.15, but less than 0.30
- C) At least 0.30, but less than 0.45
- D) At least 0.45, but less than 0.60
- E) At least 0.60

## Explanation

Using $\text{AIC} = 2p - 2\,\text{LL}$ (with $p$ = number of parameters including the intercept), each actuary chooses the size with lowest AIC among the models their algorithm can reach.

**Actuary A (best subset)** takes the highest-$R^2$ model at each size. The best-per-size AIC values are minimized at the 2-predictor model $(I, 1, 5)$: $p = 3$, LL $= 3.0$, so $\text{AIC}_A = 6 - 6 = 0$.

**Actuary B (forward stepwise)** is constrained to build on previous selections. It enters variable 2 first (highest single-variable $R^2$), then 5, then 4, so the best model it reaches is $(I, 2, 5)$: $p = 3$, LL $= 2.9$, giving $\text{AIC}_B = 6 - 5.8 = 0.2$.

$$|\text{AIC}_A - \text{AIC}_B| = |0 - 0.2| = 0.2.$$

This falls in $[0.15, 0.30)$, so the answer is **B**.
