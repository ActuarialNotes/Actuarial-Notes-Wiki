A least squares regression model is fit to a data set. The resulting predictors and their standard errors are shown below:

| | Fitted | Standard Error |
|---|--------|----------------|
| $\hat\beta_0$ | 1.00 | 0.05 |
| $\hat\beta_1$ | 2.50 | 0.15 |
| $\hat\beta_2$ | -0.50 | 0.20 |
| $\hat\beta_3$ | 2.00 | 0.80 |

A second model is fit to the same data using only the $3^{\text{rd}}$ independent variable:

| | Fitted | Standard Error |
|---|--------|----------------|
| $\hat\beta_0$ | 1.25 | 0.08 |
| $\hat\beta_3$ | 2.20 | 0.75 |

Calculate variance inflation factor of $\hat\beta_3$.

- A) Less than 1.00
- B) At least 1.00, but less than 1.05
- C) At least 1.05, but less than 1.10
- D) At least 1.10, but less than 1.15
- E) At least 1.15

## Explanation

The variance inflation factor measures how much the variance of $\hat\beta_3$ is inflated in the full model (with collinear predictors) relative to a model where $X_3$ is uncorrelated with the other predictors. In the simple regression using only $X_3$ there is no collinearity, so its standard error reflects the "no inflation" baseline. Holding the error variance comparable,

$$\text{VIF}_3 = \left(\frac{\text{SE}_{\text{full}}(\hat\beta_3)}{\text{SE}_{\text{simple}}(\hat\beta_3)}\right)^2 = \left(\frac{0.80}{0.75}\right)^2 = (1.0667)^2 = 1.138.$$

This falls in $[1.10, 1.15)$, so the answer is **D**.
