An electronic system contains three cooling components that operate independently. The probability of each component's failure is 0.05. The system will overheat if and only if at least two components fail.

Calculate the probability that the system will overheat.

- A) 0.007
- B) 0.045
- C) 0.098
- D) 0.135
- E) 0.143

## Explanation

The number of components that fail, $N$, has a binomial distribution with $n=3$ and $p=0.05$.
The system overheats if $N \ge 2$ (so $N=2$ or $N=3$).

$$P(N \ge 2) = P(N=2) + P(N=3)$$
$$= \binom{3}{2}(0.05)^2(0.95)^1 + \binom{3}{3}(0.05)^3(0.95)^0$$
$$= 3(0.0025)(0.95) + 1(0.000125)(1)$$
$$= 0.007125 + 0.000125 = 0.00725 \approx 0.007$$
