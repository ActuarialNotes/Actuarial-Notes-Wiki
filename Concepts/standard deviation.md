# standard deviation

## Definition

The ==standard deviation== of a random variable $X$ measures spread in the same units as $X$:

$$ \sigma = \sqrt{\text{Var}(X)} $$

> [!example]- <u>Example</u>
> Continuing from the variance example where $\text{Var}(X) = 0.81$:
>
> $\sigma = \sqrt{0.81} = 0.9$
>
> This means the typical deviation from the mean $\mu = 2.7$ is about $0.9$ units. The interval $\mu \pm \sigma = [1.8, 3.6]$ captures the values $x = 2$ and $x = 3$, which together have probability $0.3 + 0.4 = 0.7$.
