The cumulative distribution function of a random variable, $X$, is given by a point mass at 0 with probability 0.5, and increases linearly from 0.5 to 1.0 between $x=2$ and $x=3$. For all other values, the CDF is constant.
*(Note: This represents a CDF where $F(x) = 0$ for $x<0$, $F(x) = 0.5$ for $0 \le x < 2$, $F(x) = 0.5x - 0.5$ for $2 \le x < 3$, and $F(x) = 1$ for $x \ge 3$.)*

Calculate $E[X]$.

- A) 0.00
- B) 0.50
- C) 1.00
- D) 1.25
- E) 2.50

## Explanation

This is a mixed distribution. There is a discrete point mass at $x=0$ with probability $0.5$.
From $x=2$ to $x=3$, the distribution is continuous. The density function is the derivative of the CDF in this region:
$$f(x) = \frac{d}{dx} (0.5x - 0.5) = 0.5, \quad 2 \le x < 3$$

The expected value is the sum of the discrete part and the continuous part:
$$E[X] = 0 \cdot P(X=0) + \int_{2}^{3} x f(x) dx$$
$$E[X] = 0(0.5) + \int_{2}^{3} 0.5x dx$$
$$E[X] = 0 + 0.5 \left[ \frac{x^2}{2} \right]_2^3 = 0.5 \left( \frac{9}{2} - \frac{4}{2} \right) = 0.5(2.5) = 1.25$$
