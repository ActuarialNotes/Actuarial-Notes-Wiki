An investor pays 100,000 today for a 4-year investment that returns cash flows of 60,000 at the end of each of years 3 and 4. The cash flows can be reinvested at 4.0% per annum effective.

Using an annual effective interest rate of 5.0%, calculate the net present value of this investment today.

- A) –1,398
- B) –699
- C) 699
- D) 1,398
- E) 2,629

## Explanation

The first cash flow of 60,000 at time 3 earns interest for one year at 4%, producing $60,000(1.04) = 62,400$ at time 4. Combined with the second cash flow, the investment returns $62,400 + 60,000 = 122,400$ at time 4.

The present value of this combined cash flow is:

$$122,400(1.05)^{-4} = 100,699$$

The net present value is $100,699 - 100,000 = 699$.
