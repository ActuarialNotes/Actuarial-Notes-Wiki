A profile of the investments owned by an agent's clients follows:

1. 228 own annuities.
2. 220 own mutual funds.
3. 98 own life insurance and mutual funds.
4. 93 own annuities and mutual funds.
5. 16 own annuities, mutual funds, and life insurance.
6. 45 more clients own only life insurance than own only annuities.
7. 290 own only one type of investment.

Calculate the agent's total number of clients who own at least one of these three products.

- A) 455
- B) 495
- C) 496
- D) 500
- E) 516

## Explanation

Let: only A = $a$, only M = $m$, only L = $l$, only A\&M = $p_{am}$, only A\&L = $p_{al}$, only L\&M = $p_{lm}$, all = 16.

From (iv): $p_{am} + 16 = 93 \Rightarrow p_{am} = 77$. From (iii): $p_{lm} + 16 = 98 \Rightarrow p_{lm} = 82$.
From (i): $a + 77 + p_{al} + 16 = 228 \Rightarrow a + p_{al} = 135$.
From (ii): $m + 77 + 82 + 16 = 220 \Rightarrow m = 45$.
From (vii): $a + 45 + l = 290 \Rightarrow a + l = 245$.
From (vi): $l - a = 45$.

Solving: $2l = 290 \Rightarrow l = 145$, $a = 100$, $p_{al} = 35$.

$$\text{Total} = 100 + 45 + 145 + 77 + 35 + 82 + 16 = \mathbf{500}$$
