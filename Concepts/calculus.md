# Calculus

## Definition

==Calculus== is the branch of mathematics focused on limits, derivatives, integrals, and infinite series. Key prerequisite topics for actuarial exams include differentiation, integration, multivariate calculus, and Taylor series.

$$ \frac{d}{dx}\left[\int_a^x f(t)\,dt\right] = f(x) $$

> [!example]- <u>Example</u>
> Find the expected value of a continuous random variable $X$ with pdf $f(x) = 2x$ on $[0,1]$.
>
> $$E[X] = \int_0^1 x \cdot f(x)\,dx = \int_0^1 x \cdot 2x\,dx = \int_0^1 2x^2\,dx = \left[\frac{2x^3}{3}\right]_0^1 = \frac{2}{3}$$
>
> This demonstrates how integration is used to compute expected values, a fundamental operation in actuarial science.
