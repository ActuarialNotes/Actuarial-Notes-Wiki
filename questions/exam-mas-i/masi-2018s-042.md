You are given the following correlograms (ACF plots) for three separate data sets, each with $n$ observations:

![Three correlograms: Data Set I has a spike at lag 1 and another near lag 7 just above the upper bound with other lags inside the bounds; Data Set II has large slowly-decaying spikes at early lags well outside the bounds; Data Set III has all spikes inside the bounds](https://raw.githubusercontent.com/ActuarialNotes/Actuarial-Notes-Wiki/main/Media/Attachments/masi-2018s-q42-correlograms.png)

The dotted lines in each correlogram correspond to the values: $\left(-\dfrac{1}{n} \pm \dfrac{2}{\sqrt{n}}\right)$.

Determine which of the above data sets exhibit statistically significant autocorrelation.

- A) I only
- B) II only
- C) III only
- D) I, II and III
- E) The answer is not given by (A), (B), (C), or (D)

## Explanation

Autocorrelation is statistically significant when ACF values (beyond lag 0) fall **outside** the dotted significance bounds $\left(-\frac{1}{n} \pm \frac{2}{\sqrt{n}}\right)$.

- **Data Set I:** has spikes (lag 1 and near lag 7) that exceed the upper bound — significant autocorrelation present.
- **Data Set II:** has large early-lag spikes clearly outside the bounds — significant autocorrelation present.
- **Data Set III:** all spikes lie within the bounds — consistent with white noise, no significant autocorrelation.

Significant autocorrelation is present in **Data Sets I and II** (but not III). That combination is not offered in choices A–D, so the answer is **E**.
