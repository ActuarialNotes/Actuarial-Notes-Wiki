An urn contains 10 balls: 4 red and 6 blue. A second urn contains 16 red balls and an unknown number of blue balls. A single ball is drawn from each urn. The probability that both balls are the same color is 0.44.

Calculate the number of blue balls in the second urn.

- A) 4
- B) 20
- C) 24
- D) 44
- E) 64

## Explanation

For $i = 1, 2$, let $R_i$ = event that a red ball is drawn from urn $i$ and $B_i$ = event that a blue ball is drawn from urn $i$. Let $x$ be the number of blue balls in urn 2.

$$0.44 = \Pr[(R_1 \cap R_2) \cup (B_1 \cap B_2)] = \Pr[R_1]\Pr[R_2] + \Pr[B_1]\Pr[B_2]$$

$$= \frac{4}{10}\left(\frac{16}{x+16}\right) + \frac{6}{10}\left(\frac{x}{x+16}\right)$$

Therefore,

$$2.2 = \frac{32}{x+16} + \frac{3x}{x+16} = \frac{3x+32}{x+16}$$

$$2.2x + 35.2 = 3x + 32 \implies 0.8x = 3.2 \implies x = 4$$
