The joint probability density function of $X$ and $Y$ is
$$f(x,y) = 8xy, \quad 0 < y < x < 1$$

Calculate $E[Y \mid X = x]$.

- A) $x/3$
- B) $2x/3$
- C) $x/2$
- D) $x$
- E) $2x/5$

## Explanation

The marginal density of $X$:

$$f_X(x) = \int_0^x 8xy\,dy = 4x^3$$

Conditional density:

$$f_{Y|X}(y|x) = \frac{8xy}{4x^3} = \frac{2y}{x^2}, \quad 0 < y < x$$

$$E[Y|X=x] = \int_0^x y \cdot \frac{2y}{x^2}\,dy = \frac{2}{x^2}\cdot\frac{x^3}{3} = \frac{2x}{3}$$
