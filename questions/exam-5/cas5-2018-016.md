An insurance company with a book of business (Book A) has recently acquired a smaller book of business (Book B) in the same state and line of business. Given the following as of December 31, 2017:

**Book A**

| Accident Year | Reported Claims ($000s) as of (months) | | | | Calendar Year | Earned Premium ($000s) |
|---|---|---|---|---|---|---|
| | 12 | 24 | 36 | 48 | | |
| 2014 | 55,000 | 92,000 | 112,500 | 123,700 | 2014 | 175,200 |
| 2015 | 54,800 | 92,600 | 111,100 | | 2015 | 179,400 |
| 2016 | 57,000 | 94,400 | | | 2016 | 182,800 |
| 2017 | 62,600 | | | | 2017 | 184,200 |

- 75%: Book A expected claims ratio
- 1.06: 48-to-ultimate reported claim development factor

**Book B**

| Accident Year | Reported Claims ($000s) as of (months) | | | | Calendar Year | Earned Premium ($000s) |
|---|---|---|---|---|---|---|
| | 12 | 24 | 36 | 48 | | |
| 2014 | 2,600 | 5,900 | 6,700 | 7,500 | 2014 | 7,500 |
| 2015 | 3,500 | 4,300 | 6,000 | | 2015 | 9,700 |
| 2016 | 2,600 | 2,700 | | | 2016 | 11,000 |
| 2017 | 4,400 | | | | 2017 | 13,900 |

## Part a (0.75 points)

Calculate the Book A ultimate claims for accident year 2015 using the reported development technique.

### Answer

$129,543 thousand (approximately $129,495–$129,543 thousand depending on rounding)

### Explanation

**Step 1: Calculate age-to-age factors (LDFs)**

| | 12–24 | 24–36 | 36–48 |
|---|---|---|---|
| AY 2014 | 92,000 / 55,000 = 1.6727 | 112,500 / 92,000 = 1.2228 | 123,700 / 112,500 = 1.0996 |
| AY 2015 | 92,600 / 54,800 = 1.6898 | 111,100 / 92,600 = 1.1997 | |
| AY 2016 | 94,400 / 57,000 = 1.6561 | | |
| **Average** | **1.6729** | **1.2113** | **1.1** |

(Average of available data for each development period)

**Step 2: Build cumulative development factors (CDFs) to ultimate**

$$\text{CDF}_{48\text{-Ult}} = 1.06 \quad \text{(given)}$$

$$\text{CDF}_{36\text{-Ult}} = 1.1 \times 1.06 = 1.166$$

$$\text{CDF}_{24\text{-Ult}} = 1.2113 \times 1.166 = 1.412$$

$$\text{CDF}_{12\text{-Ult}} = 1.6729 \times 1.412 = 2.362$$

**Step 3: Apply CDF for AY 2015 (currently at 36 months)**

$$\text{AY 2015 Ultimate Claims} = 111{,}100 \times 1.166 = \$129{,}543 \text{ thousand}$$

### Examiner Report

Candidates were expected to calculate Book A ultimate losses for accident year 2015 using the reported development technique, including calculation of age-to-age and cumulative development factors.

Common mistakes include:
- Calculating ultimate losses for an accident year other than 2015
- Omitting the 48-to-ultimate reported development factor
- Using nonadjacent columns of the loss development triangle to calculate age-to-age factors

## Part b (0.5 points)

Calculate the Book A ultimate claims for accident year 2016 using the Bornhuetter-Ferguson technique.

### Answer

$134,388 thousand (approximately)

### Explanation

**Step 1: Determine the CDF for AY 2016 (currently at 24 months)**

From Part a: $\text{CDF}_{24\text{-Ult}} = 1.412$

**Step 2: Calculate percent unreported**

$$\% \text{ Unreported} = 1 - \frac{1}{\text{CDF}_{24\text{-Ult}}} = 1 - \frac{1}{1.412} = 0.2919 = 29.19\%$$

**Step 3: Calculate expected claims**

$$\text{Expected Claims}_{2016} = \text{Earned Premium} \times \text{ECR} = 182{,}800 \times 0.75 = 137{,}100$$

**Step 4: Apply Bornhuetter-Ferguson formula**

$$\text{Ultimate}_{2016} = \text{Reported} + \% \text{Unreported} \times \text{Expected Claims}$$

$$= 94{,}400 + 0.2919 \times 137{,}100 = 94{,}400 + 40{,}019 = \$134{,}419 \text{ thousand}$$

(Minor rounding differences produce answers in the range $134,388–$134,419 depending on decimal precision kept.)

### Examiner Report

Candidates were expected to calculate ultimate losses for accident year 2016 using the Bornhuetter-Ferguson technique, including calculation of the percent unreported and expected losses.

Common mistakes include:
- Using an age-to-age factor to calculate the percent unreported instead of the cumulative development factor
- Calculating ultimate losses for an accident year other than 2016
- Using the percent reported instead of the percent unreported
- Calculating an expected claim ratio instead of using the given ECR of 75%

## Part c (0.25 points)

Calculate the Book A ultimate claims for accident year 2017 using the expected claims technique.

### Answer

$138,150 thousand

### Explanation

The expected claims technique uses the expected claims ratio applied to earned premium, without crediting any reported development:

$$\text{Ultimate}_{2017} = \text{ECR} \times \text{Earned Premium}_{2017} = 0.75 \times 184{,}200 = \$138{,}150 \text{ thousand}$$

### Examiner Report

Candidates were expected to calculate ultimate losses using the expected claims technique.

A common error was calculating an expected claim ratio instead of using the given ECR of 75%.

## Part d (0.5 points)

Recommend an approach for Book B in accident year 2015 without performing any calculations. Justify all assumptions.

### Explanation

Book B is a small and volatile book of business. Any technique that relies solely on Book B's own historical development pattern would be unreliable, because the sparse data makes age-to-age factors highly unstable and susceptible to distortion from a single large claim.

**Recommended approach:** Use Book A's development factors (or expected claims ratio) to estimate Book B's ultimates, since A and B operate in the same state and the same line of business. This shared operating environment means they are likely subject to the same claim development patterns, legal environment, and cost trends, making Book A's experience a credible and appropriate benchmark for Book B.

For example:
- **Development technique using Book A's LDFs:** Apply the CDF derived from Book A to Book B's AY 2015 reported claims at 36 months. Justified because the same development pattern should apply given the same state and LOB.
- **Bornhuetter-Ferguson using Book A:** Use Book A's CDF (for the percent unreported) and Book A's expected claims ratio as the a priori for Book B. This blends Book B's actual emergence with stable external expectations.
- **Expected claims technique using Book A:** Apply Book A's ECR directly to Book B's earned premium. Appropriate given Book B's volatility and small size, which make its own development data unreliable.

Any of the above techniques is acceptable provided the answer: (1) notes that Book B's own data is insufficient for a standalone development approach, and (2) justifies using Book A's parameters on the basis that A and B share the same state and line of business.

### Examiner Report

Candidates were expected to recognize that Book B is small and volatile, so any development technique that relies on Book B's own historical development pattern is inappropriate. Candidates were expected to recommend and justify a specific alternative approach. For recommended techniques using Book A, candidates were expected to state that Book A is appropriate to use since A and B operate in the same state and line of business.

Common mistakes include:
- Recommending a technique using Book B's historical development pattern
- Providing a recommendation with no justification
- Recommending a technique using Book A without stating why Book A is appropriate to use
- Recommending use of an "appropriate" expected claims ratio for Book B without specifying how to select one (e.g., using Book A's ratio or industry data)
- Attempting to diagnose a change in case reserve adequacy and recommending a Berquist-Sherman adjustment
