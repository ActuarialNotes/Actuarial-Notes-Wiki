Given the following information for the rating variables vehicle class, driver type, and territory:

**Exposures by Vehicle Class and Territory:**

| Vehicle Class | Territory 1 | Territory 2 | Territory 3 |
|---------------|-------------|-------------|-------------|
| A | 30 | 15 | 200 |
| B | 80 | 22 | 104 |
| C | 20 | 80 | 60 |

**Exposures by Driver Type and Territory:**

| Driver Type | Territory 1 | Territory 2 | Territory 3 |
|-------------|-------------|-------------|-------------|
| X | 50 | 45 | 140 |
| Y | 80 | 72 | 224 |

**Losses (\$000s) by Vehicle Class and Territory:**

| Vehicle Class | Territory 1 | Territory 2 | Territory 3 |
|---------------|-------------|-------------|-------------|
| A | 30 | 15 | 200 |
| B | 100 | 33 | 135 |
| C | 30 | 200 | 105 |

**Losses (\$000s) by Driver Type and Territory:**

| Driver Type | Territory 1 | Territory 2 | Territory 3 |
|-------------|-------------|-------------|-------------|
| X | 40 | 65 | 115 |
| Y | 120 | 180 | 325 |

- All segments are fully credible.
- Vehicle class A, Driver type X, and Territory 1 are used as the bases.

Propose rating factors for all three variables using a univariate method, considering any exposure correlation.

## Part a (2.75 points)

Propose rating factors for vehicle class, driver type, and territory using a univariate method that accounts for exposure correlations among the rating variables.

### Explanation

**Step 1: Check for exposure correlation between driver type and territory.**

The driver type mix by territory:

| Territory | Driver X Exp | Driver Y Exp | Total | % X | % Y |
|-----------|-------------|-------------|-------|-----|-----|
| 1 | 50 | 80 | 130 | 38.5% | 61.5% |
| 2 | 45 | 72 | 117 | 38.5% | 61.5% |
| 3 | 140 | 224 | 364 | 38.5% | 61.5% |

Driver type mix is identical across all territories, so **driver type and territory are not correlated**. We can analyze driver type and territory independently (no adjustment needed between these two).

**Step 2: Compute unadjusted (simple univariate) factors for driver type.**

Total exposures and losses by driver type (summing across territories):

| Driver Type | Exposures | Losses (\$000s) | Pure Premium |
|-------------|-----------|-----------------|--------------|
| X | 50+45+140 = 235 | 40+65+115 = 220 | 220/235 = 936.17 |
| Y | 80+72+224 = 376 | 120+180+325 = 625 | 625/376 = 1,662.23 |

Driver type relativities (base = X):

$$\text{Factor}_X = 1.000$$

$$\text{Factor}_Y = \frac{1{,}662.23}{936.17} = 1.776$$

**Step 3: Compute unadjusted (simple univariate) factors for vehicle class.**

Total exposures and losses by vehicle class (summing across territories):

| Vehicle Class | Exposures | Losses (\$000s) | Pure Premium |
|---------------|-----------|-----------------|--------------|
| A | 30+15+200 = 245 | 30+15+200 = 245 | 245/245 = 1,000 |
| B | 80+22+104 = 206 | 100+33+135 = 268 | 268/206 = 1,300.97 |
| C | 20+80+60 = 160 | 30+200+105 = 335 | 335/160 = 2,093.75 |

Vehicle class relativities (base = A):

$$\text{Factor}_A = 1.000, \quad \text{Factor}_B = \frac{1{,}300.97}{1{,}000} = 1.301, \quad \text{Factor}_C = \frac{2{,}093.75}{1{,}000} = 2.094$$

**Step 4: Check for exposure correlation between vehicle class and territory.**

Vehicle class A mix by territory: 30/130=23.1%, 15/117=12.8%, 200/364=54.9% — the mix differs by territory, so **vehicle class and territory are correlated**. We must adjust territory exposures for vehicle class mix before computing territory factors.

**Step 5: Adjust territory exposures for vehicle class mix (to isolate territory effect).**

Compute adjusted exposures for each territory by reweighting vehicle class exposures using the vehicle class factors:

$$\text{Adj Exp}_{t} = \sum_{\text{class}} \text{Exp}_{\text{class},t} \times \text{Factor}_{\text{class}}$$

| Territory | Adj Exposure | Losses (\$000s) |
|-----------|-------------|-----------------|
| 1 | 30(1.000)+80(1.301)+20(2.094) = 30+104.08+41.88 = 175.96 | 160 |
| 2 | 15(1.000)+22(1.301)+80(2.094) = 15+28.62+167.52 = 211.14 | 248 |
| 3 | 200(1.000)+104(1.301)+60(2.094) = 200+135.30+125.64 = 460.94 | 440 |

Pure premiums and territory relativities (base = Territory 1):

$$\text{PP}_1 = \frac{160}{175.96} = 909.3, \quad \text{PP}_2 = \frac{248}{211.14} = 1{,}174.6, \quad \text{PP}_3 = \frac{440}{460.94} = 954.6$$

$$\text{Factor}_1 = 1.000, \quad \text{Factor}_2 = \frac{1{,}174.6}{909.3} = 1.292, \quad \text{Factor}_3 = \frac{954.6}{909.3} = 1.050$$

**Summary of proposed rating factors:**

| Variable | Level | Factor |
|----------|-------|--------|
| Vehicle Class | A | 1.000 |
| Vehicle Class | B | 1.301 |
| Vehicle Class | C | 2.094 |
| Driver Type | X | 1.000 |
| Driver Type | Y | 1.776 |
| Territory | 1 | 1.000 |
| Territory | 2 | 1.292 |
| Territory | 3 | 1.050 |

Note: Because driver type is not correlated with territory, no exposure adjustment is needed for that pairing. Vehicle class IS correlated with territory, so adjusted exposures were used for the territory analysis.

### Examiner Report

Candidates were expected to recognize exposure correlations across rating variables and demonstrate how to adjust for correlations using univariate analysis.

Common mistakes included: performing univariate analysis on rating variables while ignoring exposure correlations; using loss tables rather than exposure tables when developing exposure adjustment weights; using loss relativities rather than pure premium relativities in the analysis; applying incorrect weights for exposure adjustment; and incorrect application of sequential analysis.

Note from the Syllabus and Examination Committee: There was an inconsistency in the question, where the total losses for Territory 2 across vehicle class and driver type were not equal. This was unintended. The question was still possible to answer using either vehicle class or driver type for the territory analysis.
