An insurance agent's files reveal the following facts about his policyholders:

1. 243 own auto insurance.
2. 207 own homeowner insurance.
3. 55 own life insurance and homeowner insurance.
4. 96 own auto insurance and homeowner insurance.
5. 32 own life insurance, auto insurance and homeowner insurance.
6. 76 more clients own only auto insurance than only life insurance.
7. 270 own only one of these three insurance products.

Calculate the total number of the agent's policyholders who own at least one of these three insurance products.

- A) 389
- B) 407
- C) 423
- D) 448
- E) 483

## Explanation

Let: only A = $a$, only H = $h$, only L = $l$, only A\&H = $p$, only A\&L = $q$, only L\&H = $r$, all three = 32.

From (iii): $r + 32 = 55 \Rightarrow r = 23$. From (iv): $p + 32 = 96 \Rightarrow p = 64$.
From (i): $a + p + q + 32 = 243 \Rightarrow a + q = 147$.
From (ii): $h + p + r + 32 = 207 \Rightarrow h = 88$.
From (vii): $a + h + l = 270 \Rightarrow a + l = 182$.
From (vi): $a - l = 76$.

Solving: $2a = 258 \Rightarrow a = 129$, $l = 53$, $q = 147 - 129 = 18$.

$$\text{Total} = 129 + 88 + 53 + 64 + 18 + 23 + 32 = \mathbf{407}$$
