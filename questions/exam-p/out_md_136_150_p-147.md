In a certain group of cancer patients, each patient's cancer is classified in exactly one of the following five stages: stage 0, stage 1, stage 2, stage 3, or stage 4.

i) 75% of the patients in the group have stage 2 or lower.
ii) 80% of the patients in the group have stage 1 or higher.
iii) 80% of the patients in the group have stage 0, 1, 3, or 4.

One patient from the group is randomly selected. Calculate the probability that the selected patient's cancer is stage 1.

- A) 0.20
- B) 0.25
- C) 0.35
- D) 0.48
- E) 0.65

## Explanation

Let $p_i$ represent the probability that the patient's cancer is in stage $i$, for $i = 0, 1, 2, 3, 4$.
The sum of all probabilities must equal 1: $p_0 + p_1 + p_2 + p_3 + p_4 = 1$.

From the given facts:
i) $p_0 + p_1 + p_2 = 0.75$
ii) $p_1 + p_2 + p_3 + p_4 = 0.80$
iii) $p_0 + p_1 + p_3 + p_4 = 0.80$

From (ii), we can find $p_0$ because $p_0 = 1 - (p_1 + p_2 + p_3 + p_4) = 1 - 0.80 = 0.20$.
From (iii), we can find $p_2$ because $p_2 = 1 - (p_0 + p_1 + p_3 + p_4) = 1 - 0.80 = 0.20$.

Substitute $p_0$ and $p_2$ into equation (i):
$$p_0 + p_1 + p_2 = 0.75$$
$$0.20 + p_1 + 0.20 = 0.75$$
$$0.40 + p_1 = 0.75$$
$$p_1 = 0.35$$
