George and Paul play a betting game. Each chooses an integer from 1 to 20 (inclusive) at random. If the two numbers differ by more than 3, George wins the bet. Otherwise, Paul wins the bet.

Calculate the probability that Paul wins the bet.

- A) 0.27
- B) 0.32
- C) 0.40
- D) 0.48
- E) 0.66

## Explanation

Let $X$ and $Y$ be the numbers chosen by George and Paul. The total number of outcomes is $20 \times 20 = 400$.
Paul wins if $|X - Y| \le 3$.

We count the number of outcomes that satisfy this condition:
* If $X=1$, $Y$ can be 1, 2, 3, 4 $\implies 4$ outcomes.
* If $X=2$, $Y$ can be 1, 2, 3, 4, 5 $\implies 5$ outcomes.
* If $X=3$, $Y$ can be 1, 2, 3, 4, 5, 6 $\implies 6$ outcomes.
* If $X=4$ through $17$, $Y$ can be $X-3, \dots, X+3$ $\implies 7$ outcomes each ($14 \times 7 = 98$ outcomes).
* If $X=18$, $Y$ can be 15, 16, 17, 18, 19, 20 $\implies 6$ outcomes.
* If $X=19$, $Y$ can be 16, 17, 18, 19, 20 $\implies 5$ outcomes.
* If $X=20$, $Y$ can be 17, 18, 19, 20 $\implies 4$ outcomes.

The total number of winning outcomes for Paul is $4 + 5 + 6 + 98 + 6 + 5 + 4 = 128$.
The probability Paul wins is $\frac{128}{400} = 0.32$.
