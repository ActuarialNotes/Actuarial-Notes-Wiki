The price of a 36-year zero-coupon bond is 80% of its face value.

A second bond, with the same price, same face value, and same annual effective yield rate, offers annual coupons with the coupon rate equal to $\dfrac{4}{9}$ of the annual effective yield rate.

Calculate the number of years until maturity for the second bond.

- A) 45
- B) 54
- C) 63
- D) 72
- E) 81

## Explanation

For the zero-coupon bond: $0.8 = v^{36}$, so $i = (1/0.8)^{1/36}-1 = 0.006218$.

For the second bond, coupon rate $r = (4/9)i$. Using the premium-discount formula:
$$0.8F = Fv^n + F(r-i)a_{\overline{n}|} = F\left[v^n + \left(\frac{4}{9}i - i\right)a_{\overline{n}|}\right] = F\left[v^n - \frac{5}{9}ia_{\overline{n}|}\right]$$

$$0.8 = v^n - \frac{5}{9}(1-v^n) = \frac{4}{9}v^n + \frac{4}{9}v^n + \frac{4}{9} ... $$

Actually: $0.8 = v^n + \frac{4i-9i}{9}a_{\overline{n}|} = v^n - \frac{5i}{9}\cdot\frac{1-v^n}{i} = v^n - \frac{5}{9}(1-v^n) = \frac{14}{9}v^n - \frac{5}{9}$

$$0.8 + \frac{5}{9} = \frac{14}{9}v^n \implies v^n = \frac{9(0.8+5/9)}{14} = \frac{9 \times 1.3556}{14} = 0.8715$$

Using calculator with $PV=0.8$, $I/Y=0.6218$, $PMT=0.002763$, $FV=1$: $n=72$.
