Consider two 30-year bonds with the same purchase price. Each has an annual coupon rate of 5% paid semiannually and a par value of 1000.

The first bond has an annual nominal yield rate of 5% compounded semiannually, and a redemption value of 1200.

The second bond has an annual nominal yield rate of $j$ compounded semiannually, and a redemption value of 800.

Calculate $j$.

- A) 2.20%
- B) 2.34%
- C) 3.53%
- D) 4.40%
- E) 4.69%

## Explanation

**First bond** (2.5% per period, 60 periods):

$$P = 25a_{\overline{60}|0.025} + 1200(1.025)^{-60} = 772.72 + 272.74 = 1{,}045.46$$

**Second bond** has the same price 1{,}045.46. Coupon = 25 per period, redemption = 800, 60 periods, find $j/2$:

$$1{,}045.46 = 25a_{\overline{60}|j/2} + 800(1+j/2)^{-60}$$

Using a financial calculator: $j/2 = 2.2\%$, so $j = 4.4\%$.
