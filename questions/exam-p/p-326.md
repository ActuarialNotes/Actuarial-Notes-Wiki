The monthly commission that an agent earns is modeled by a random variable $X$ with density function
$$f(x) = \frac{1}{20}e^{-x/20}, \quad x > 0$$

Calculate the probability that the commission the agent earns in a month is within 0.5 standard deviations of $E(X)$.

- A) 0.34
- B) 0.38
- C) 0.50
- D) 0.68
- E) 0.95

## Explanation

$X \sim \text{Exp}(\text{mean }20)$, so $E[X]=20$ and $\text{SD}(X)=20$.

$P(|X-20| \leq 0.5 \times 20) = P(|X-20| \leq 10) = P(10 \leq X \leq 30)$

$$= e^{-10/20}-e^{-30/20} = e^{-0.5}-e^{-1.5}$$

$$\approx 0.6065-0.2231 = \mathbf{0.3834 \approx 0.38}$$
