A model is created with the following form:

$$f(x) = \beta_0 + \beta_1 x + \beta_2 x^2 + \beta_3 x^3 + \beta_4 (x - \xi)_+^3$$

where:

$$(x - \xi)_+^3 = (x - \xi)^3 \text{ if } x > \xi, \text{ and } 0 \text{ otherwise}$$

You are given the following statements:

- I. $f(x)$ is continuous at $\xi$
- II. $f'(x)$ is continuous at $\xi$
- III. $f''(x)$ is continuous at $\xi$

Determine which of the above statements are true.

- A) I only
- B) II only
- C) III only
- D) I, II, and III
- E) The answer is not given by (A), (B), (C), or (D)

## Explanation

This is a **cubic spline** with a single knot at $\xi$: a global cubic plus the truncated power basis term $(x-\xi)_+^3$. The truncated cubic term and its first two derivatives all equal zero at $x = \xi$:

$$(x-\xi)_+^3\big|_{\xi} = 0,\quad \frac{d}{dx}(x-\xi)_+^3\big|_{\xi} = 3(x-\xi)^2\big|_{\xi} = 0,\quad \frac{d^2}{dx^2}(x-\xi)_+^3\big|_{\xi} = 6(x-\xi)\big|_{\xi} = 0.$$

So $f$, $f'$, and $f''$ are all continuous at the knot (only the *third* derivative jumps, by $6\beta_4$). All three statements are true, so the answer is **D**.
