An actuary determines the following regarding an individual auto policyholder: $P(\text{medical claim})=0.30$, $P(\text{property claim})=0.42$, $P(\text{medical or property claim})=0.60$.

Calculate the probability that the auto policyholder will file exactly one type of claim, given that the policyholder will not file both types of claims.

- A) 0.45
- B) 0.48
- C) 0.52
- D) 0.55
- E) 0.60

## Explanation

$P(M \cap P) = 0.30+0.42-0.60 = 0.12$

$P(\text{exactly one}) = P(M \cup P)-P(M \cap P) = 0.60-0.12 = 0.48$

$P(\text{not both}) = 1-0.12 = 0.88$

$$P(\text{exactly one}\mid\text{not both}) = \frac{0.48}{0.88} = \frac{6}{11} \approx \mathbf{0.545 \approx 0.55}$$
