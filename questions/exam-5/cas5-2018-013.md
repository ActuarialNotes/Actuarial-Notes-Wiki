Given the following information about a home's propensity for loss:

| Size of Loss | | Loss Distribution |
|---|---|---|
| $X \leq$ | $400,000 | 50.0% |
| $400,000 < X \leq$ | $550,000 | 25.0% |
| $550,000 < X \leq$ | $700,000 | 10.0% |
| $700,000 < X \leq$ | $850,000 | 10.0% |
| $850,000 < X \leq$ | $1,000,000 | 2.5% |
| $1,000,000 < X \leq$ | $1,500,000 | 2.5% |
| Total: | | 100.0% |

- Expected claim frequency is 2%.
- Expected losses are uniformly distributed within each layer of loss.
- The home is valued at $1,500,000.

## Part a (1.25 points)

Calculate the rate per $1,000 of coverage for the home at the following amounts of insurance:

i. $1,500,000
ii. $800,000

### Answer

i. $5.51 per $1,000
ii. $9.95 per $1,000

### Explanation

**Part i: Amount of Insurance = $1,500,000**

Calculate average severity using the midpoint of each loss range:

$$\bar{S} = 0.50(200{,}000) + 0.25(475{,}000) + 0.10(625{,}000) + 0.10(775{,}000) + 0.025(925{,}000) + 0.025(1{,}250{,}000)$$

$$= 100{,}000 + 118{,}750 + 62{,}500 + 77{,}500 + 23{,}125 + 31{,}250 = \$413{,}125$$

Pure premium = frequency $\times$ severity = $0.02 \times 413{,}125 = \$8{,}262.50$

$$\text{Rate per \$1,000 AOI} = \frac{8{,}262.50}{1{,}500{,}000 / 1{,}000} = \frac{8{,}262.50}{1{,}500} = \$5.51$$

**Part ii: Amount of Insurance = $800,000**

Losses are capped at $800,000. The range $700,000 < X \leq \$850,000$ (10% probability) must be split: $\frac{2}{3}$ of the range falls at or below $800,000 and $\frac{1}{3}$ falls above.

- Losses in $700{,}000 < X \leq 800{,}000$: midpoint = $750{,}000$; probability = $10\% \times \frac{2}{3} = 6.67\%$
- Losses in $800{,}000 < X \leq 850{,}000$: capped at $800,000; probability = $10\% \times \frac{1}{3} = 3.33\%$
- All layers above $850,000 are also capped at $800,000: probability = $2.5\% + 2.5\% = 5\%$

$$\bar{S} = 0.50(200{,}000) + 0.25(475{,}000) + 0.10(625{,}000) + 0.0667(750{,}000) + (0.0333 + 0.05)(800{,}000)$$

$$= 100{,}000 + 118{,}750 + 62{,}500 + 50{,}000 + 66{,}667 = \$397{,}917$$

Pure premium = $0.02 \times 397{,}917 = \$7{,}958.33$

$$\text{Rate per \$1,000 AOI} = \frac{7{,}958.33}{800{,}000 / 1{,}000} = \frac{7{,}958.33}{800} = \$9.95$$

### Examiner Report

Candidates were expected to calculate the rate per amount of insurance (AOI) by computing the average severity using the midpoint of each range and the supplied loss distribution, then multiplying by frequency and dividing by the AOI (in $000s).

Common mistakes for part i:
- Using the upper or lower bound of the range instead of the midpoint
- Not multiplying by frequency
- Not dividing by AOI

Common mistakes for part ii:
- Incorrectly accounting for the capping of coverage at $800,000 by not using the correct average severity for the range $700,000–$850,000
- Omitting all layers above $800,000
- Using the upper or lower bound instead of the midpoint
- Not multiplying by frequency or not dividing by AOI

## Part b (0.5 points)

Briefly discuss a problem associated with underinsurance from the following perspectives:

i. Insured
ii. Insurer

### Explanation

**i. Insured perspective:**

If the insured is underinsured, in the case of a total loss or near-total loss, the insured will not receive enough indemnity to fully rebuild their home and will need to cover the difference out of pocket. Additionally, the insured may face coinsurance penalties that reduce payments even for partial losses.

**ii. Insurer perspective:**

If the insurer calculates rates assuming all policies are fully insured to replacement cost value, then premiums charged for underinsured homes will be inadequate to cover expected losses. This results in inequitable rates, where fully insured policyholders effectively subsidize those who are underinsured.

### Examiner Report

Candidates were expected to demonstrate a problem with underinsurance from each perspective.

Common mistakes for part i:
- Simply stating the insured would not receive the full loss payment without specifying that this applies to total or near-total losses
- Describing an issue for the insurer instead of the insured

Common mistakes for part ii:
- Stating that the insurer's profitability would be impacted without directional justification
- Stating rates would be inaccurate without specifying they would be inadequate
- Stating rates would be skewed without specifying how

## Part c (1 point)

The home is insured for $1,000,000 with a coinsurance requirement of 80%. Calculate the indemnity payments and coinsurance penalties for the following losses:

i. $800,000
ii. $1,200,000

### Answer

i. Indemnity = $666,667; Coinsurance penalty = $133,333
ii. Indemnity = $1,000,000; Coinsurance penalty = $0

### Explanation

The coinsurance apportionment ratio is:

$$a = \min\!\left(\frac{\text{Insurance Carried}}{\text{Coinsurance Requirement} \times \text{Property Value}}, 1\right) = \min\!\left(\frac{1{,}000{,}000}{0.80 \times 1{,}500{,}000}, 1\right) = \min(0.8333, 1) = 0.8333$$

The indemnity formula is: $I = \min(a \times \text{Loss}, \text{Policy Limit})$

The coinsurance penalty is: $e = \min(\text{Loss}, \text{Policy Limit}) - I$

**Part i: Loss = $800,000**

$$I = \min(0.8333 \times 800{,}000,\ 1{,}000{,}000) = \min(666{,}667,\ 1{,}000{,}000) = \$666{,}667$$

$$e = 800{,}000 - 666{,}667 = \$133{,}333$$

**Part ii: Loss = $1,200,000**

$$I = \min(0.8333 \times 1{,}200{,}000,\ 1{,}000{,}000) = \min(1{,}000{,}000,\ 1{,}000{,}000) = \$1{,}000{,}000$$

$$e = \min(1{,}200{,}000,\ 1{,}000{,}000) - 1{,}000{,}000 = 1{,}000{,}000 - 1{,}000{,}000 = \$0$$

There is no coinsurance penalty because the policy limit has been reached — the insured cannot collect more than the policy limit regardless of the apportionment ratio.

### Examiner Report

Candidates were expected to calculate the appropriate apportionment ratio and apply it to the loss to determine the indemnity payment and coinsurance penalty.

Common mistakes include:
- Using an incorrect apportionment ratio calculation
- Including the loss amount in excess of the coverage in the coinsurance penalty
