Three asset-liability cash flows are given in the following table where a positive amount is an asset cash flow and a negative amount is a liability due at the corresponding time.

| $t$ (in years) | 0 | 1 | 2 | 3 |
|---|---|---|---|---|
| X | 102,400 | −192,000 | 0 | 100,000 |
| Y | 158,400 | −342,000 | 100,000 | 100,000 |
| Z | −89,600 | 288,000 | 100,000 | −300,000 |

Determine which set of cash flows is Redington immunized for an annual effective interest rate of $i = 25\%$.

- A) X only
- B) Y only
- C) Z only
- D) X, Y, and Z
- E) The correct answer is not given by (A), (B), (C) or (D).

## Explanation

For Redington immunization at $i=25\%$, we need $h(0.25)=0$, $h'(0.25)=0$, and $h''(0.25)>0$.

**X:** $h(0.25) = 102{,}400 - 192{,}000/1.25 + 100{,}000/1.25^3 = 102{,}400 - 153{,}600 + 51{,}200 = 0$ ✓

$h'(0.25) = 192{,}000/1.25^2 - 300{,}000/1.25^4 = 122{,}880 - 122{,}880 = 0$ ✓

$h''(0.25) = -192{,}000(2)/1.25^3 + 100{,}000(3)(4)/1.25^5 = -196{,}608 + 393{,}216 = 196{,}608 > 0$ ✓

**Y:** $h'(0.25) = 342{,}000/1.25^2 - 200{,}000/1.25^3 - 300{,}000/1.25^4 = 218{,}880 - 102{,}400 - 122{,}880 = -6{,}400 \ne 0$ ✗

**Z:** $h(0.25) = -89{,}600 + 288{,}000/1.25 + 100{,}000/1.25^2 - 300{,}000/1.25^3 = -89{,}600 + 230{,}400 + 64{,}000 - 153{,}600 = 51{,}200 \ne 0$ ✗

Only **X** is Redington immunized.
