# Event

## Definition

An ==event== is a subset of the sample space. A simple event consists of a single outcome, while a compound event consists of multiple outcomes.

> [!example]- <u>Example</u>
> Let $S = \{1, 2, 3, 4, 5, 6\}$ be the sample space for rolling a single die.
> - **Simple event:** $A = \{3\}$ (rolling a 3)
> - **Compound event:** $B = \{2, 4, 6\}$ (rolling an even number)
> - **Complement:** $B^c = \{1, 3, 5\}$ (rolling an odd number)
> - **Certain event:** $S = \{1,2,3,4,5,6\}$
> - **Impossible event:** $\emptyset$
