Given the following as of December 31, 2017:

| Accident Year | Cumulative Received S&S ($000) at 12 months | at 24 months | at 36 months | at 48 months |
|---------------|----------------------------------------------|--------------|--------------|--------------|
| 2014 | 4,700 | 7,000 | 7,200 | 7,300 |
| 2015 | 4,300 | 6,600 | 6,800 | |
| 2016 | 4,300 | 6,800 | | |
| 2017 | 4,900 | | | |

| Accident Year | Cumulative Paid Claims Gross of S&S ($000) at 12 months | at 24 months | at 36 months | at 48 months |
|---------------|----------------------------------------------------------|--------------|--------------|--------------|
| 2014 | 13,500 | 16,800 | 16,800 | 16,800 |
| 2015 | 13,300 | 16,900 | 16,900 | |
| 2016 | 13,200 | 16,800 | | |
| 2017 | 12,900 | | | |

| Accident Year | Selected Ultimate Claims Gross of S&S ($000) |
|---------------|----------------------------------------------|
| 2014 | 16,800 |
| 2015 | 16,900 |
| 2016 | 16,800 |
| 2017 | 16,400 |

- There is no development beyond 48 months.

## Part a (1.5 points)

Estimate ultimate salvage and subrogation for accident year 2017 using a ratio approach.

### Answer

$6,917,520 (selecting average of AY 2014–2016 ultimate ratios) or approximately $7,958,920 (recognizing increasing trend and selecting 48-to-ultimate CDF = 1.000).

### Explanation

**Step 1: Calculate S&S to Paid Claims ratios**

$$\text{S\&S Ratio} = \frac{\text{Cumulative Received S\&S}}{\text{Cumulative Paid Claims Gross of S\&S}}$$

| Accident Year | 12 months | 24 months | 36 months | 48 months |
|---------------|-----------|-----------|-----------|-----------|
| 2014 | 0.348 | 0.417 | 0.429 | 0.435 |
| 2015 | 0.323 | 0.391 | 0.402 | |
| 2016 | 0.326 | 0.405 | | |
| 2017 | 0.380 | | | |

**Step 2: Calculate link ratios (age-to-age factors for the ratio triangle)**

| Accident Year | 12–24 | 24–36 | 36–48 |
|---------------|-------|-------|-------|
| 2014 | 1.198 | 1.029 | 1.014 |
| 2015 | 1.211 | 1.028 | |
| 2016 | 1.242 | | |

**Step 3: Select development factors**

Treating fluctuations as random, select the average:

| Period | 12–24 | 24–36 | 36–48 | 48–Ult |
|--------|-------|-------|-------|--------|
| Selected LDF | 1.217 | 1.0285 | 1.014 | 1.000 |
| CDF to Ultimate | 1.2692 | 1.0429 | 1.014 | 1.000 |

**Step 4: Develop AY 2017 to ultimate**

$$\text{AY 2017 Ultimate Ratio} = 0.380 \times 1.2692 = 0.4823$$

**Step 5: Compare to prior years' ultimate ratios**

| Accident Year | Ultimate S&S Ratio |
|---------------|-------------------|
| 2014 | 0.435 |
| 2015 | $0.402 \times 1.014 = 0.408$ |
| 2016 | $0.405 \times 1.0429 = 0.422$ |
| 2017 (developed) | 0.4823 |

AY 2017's developed ratio is relatively high compared to prior years. A reasonable selection is the average of the prior three years:

$$\text{Selected Ratio} = \frac{0.435 + 0.408 + 0.422}{3} = 0.421$$

**Step 6: Apply selected ratio to AY 2017 ultimate claims**

$$\text{Ultimate S\&S} = 0.421 \times 16{,}400{,}000 = \$6{,}917{,}520$$

*Alternatively*, if an increasing trend is observed in the ratios, one could keep the 0.4853 developed ratio (using the average of the last two years' LDFs), yielding:

$$0.4853 \times 16{,}400{,}000 = \$7{,}958{,}920$$

### Examiner Report

Candidates were expected to know how to apply the ratio approach to estimate ultimate salvage and subrogation. This involves calculating the appropriate ratios, calculating the development factors for these ratios, making an actuarially sound selection of an ultimate ratio, and using that ratio to estimate the ultimate salvage and subrogation dollars.

Common mistakes included:

- Failure to calculate and/or consider ultimate S&S ratios for years 2014–2016.
- Confusing the ratio approach for S&S with ALAE estimate methods.
- Calculating S&S ratios using ultimate gross claims instead of cumulative paid claims in the denominator.
