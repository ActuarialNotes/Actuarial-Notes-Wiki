Two ordinary least squares models were built to predict expected annual losses on Homeowners policies. Information for the two models is provided below.

**Model 1** parameters: Intercept 212; Replacement Cost (000s) 0.03; Roof Size 0.15; Precipitation Index 120; Replacement Cost (000s) x Roof Size 0.0010. Model 1 statistics: $R^2 = 0.91$, Adj $R^2 = 0.87$, MSE 31,765, AIC 25,031.

**Model 2** parameters: Intercept 315; Replacement Cost (000s) 0.02; Roof Size 0.17; Number of Bathrooms 210; Replacement Cost (000s) x Roof Size 0.0015. Model 2 statistics: $R^2 = 0.94$, Adj $R^2 = 0.89$, MSE 30,689, AIC 25,636.

Cross-validation MSE by fold:

| Fold | Model 1 | Model 2 |
|------|---------|---------|
| 1 | 33,415 | 26,666 |
| 2 | 38,741 | 38,554 |
| 3 | 32,112 | 39,662 |
| 4 | 37,210 | 36,756 |
| 5 | 29,501 | 30,303 |

You use 5-fold cross validation to select the superior of the two models.

Calculate the predicted expected annual loss for a homeowners policy with a 500,000 replacement cost, a 2,000 roof size, a 0.89 precipitation index and three bathrooms, using the selected model.

- A) Less than 1,000
- B) At least 1,000, but less than 1,500
- C) At least 1,500, but less than 2,000
- D) At least 2,000, but less than 2,500
- E) At least 2,500

## Explanation

**Model selection.** Average the cross-validation MSE across folds:

$$\text{Model 1: } \frac{33415 + 38741 + 32112 + 37210 + 29501}{5} = 34{,}195.8,$$
$$\text{Model 2: } \frac{26666 + 38554 + 39662 + 36756 + 30303}{5} = 34{,}388.2.$$

Model 1 has the lower average CV error, so it is selected.

**Prediction with Model 1** (replacement cost $500$ in 000s, roof size $2000$, precipitation index $0.89$; number of bathrooms is not in Model 1):

$$\hat{y} = 212 + 0.03(500) + 0.15(2000) + 120(0.89) + 0.0010(500 \times 2000)$$
$$= 212 + 15 + 300 + 106.8 + 1000 = 1633.8.$$

This falls in $[1{,}500, 2{,}000)$, so the answer is **C**.
