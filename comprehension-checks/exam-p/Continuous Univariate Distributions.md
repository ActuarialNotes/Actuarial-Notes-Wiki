For a continuous random variable X with density f, which statement is true?

- A) P(X = c) = 0 for any single value c
- B) f(x) can never exceed 1, since it is a probability
- C) f(x) equals the probability that X takes the value x
- D) Probabilities are found by summing f over the possible values

<!-- rationale: 1: density can exceed 1 · 2: confuses density with probability · 3: continuous variables integrate, not sum -->
