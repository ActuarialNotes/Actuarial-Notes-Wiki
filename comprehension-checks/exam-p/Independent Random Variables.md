Which statement about independent random variables is correct?

- A) If X and Y are independent, then Cov(X, Y) = 0
- B) If Cov(X, Y) = 0, then X and Y must be independent
- C) Independence requires E[XY] = 0
- D) Independent variables always share the same distribution
