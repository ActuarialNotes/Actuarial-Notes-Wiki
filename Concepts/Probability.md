# Probability

## Definition

==Probability== is a set function $P: \mathcal{F} \to [0,1]$ that assigns a measure of likelihood to each event satisfying the axioms of probability. For equally likely outcomes, the classical definition gives:

$$ P(A) = \frac{|A|}{|S|} $$

> [!example]- <u>Example</u>
> A bag contains 5 red, 3 blue, and 2 green marbles (10 total). Each marble is equally likely to be drawn.
> - $P(\text{red}) = \frac{5}{10} = 0.5$
> - $P(\text{blue}) = \frac{3}{10} = 0.3$
> - $P(\text{green}) = \frac{2}{10} = 0.2$
> - $P(\text{red or blue}) = \frac{5+3}{10} = 0.8$
>
> Note that $P(\text{red}) + P(\text{blue}) + P(\text{green}) = 1$, consistent with the axioms.
