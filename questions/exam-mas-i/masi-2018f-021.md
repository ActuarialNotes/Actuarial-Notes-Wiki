Two independent populations X and Y have the following density functions:

- $f(x) = \lambda x^{\lambda-1}$ for $0 < x < 1$
- $g(y) = \mu y^{\mu-1}$ for $0 < y < 1$

$X_1, X_2, \ldots, X_n$ and $Y_1, Y_2, \ldots, Y_m$ are random samples of sizes $n$ and $m$, from X and Y, respectively.

You want to perform the following hypothesis test:

- $H_0: \lambda = 2;\ \mu = 4$
- $H_1: \lambda = 3;\ \mu = 8$

Determine the form of the best critical region for this test, using the Neyman-Pearson lemma.

- A) $\sum_{i=1}^{n} \ln X_i + 4\sum_{i=1}^{m} \ln Y_i \le c$
- B) $\sum_{i=1}^{n} \ln X_i + 4\sum_{i=1}^{m} \ln Y_i \ge c$
- C) $4\sum_{i=1}^{n} \ln X_i - \sum_{i=1}^{m} \ln Y_i \le c$
- D) $4\sum_{i=1}^{n} \ln X_i - \sum_{i=1}^{m} \ln Y_i \ge c$
- E) The answer is not given by (A), (B), (C), or (D)

## Explanation

By the Neyman–Pearson lemma, the best critical region rejects $H_0$ when the likelihood ratio $L(H_1)/L(H_0)$ is large. The per-observation ratios are

$$\frac{f(x;\lambda=3)}{f(x;\lambda=2)} = \frac{3x^2}{2x} = \tfrac{3}{2}x, \qquad \frac{g(y;\mu=8)}{g(y;\mu=4)} = \frac{8y^7}{4y^3} = 2y^4.$$

So the likelihood ratio is

$$\Lambda = \left(\tfrac{3}{2}\right)^n 2^m \left(\prod x_i\right)\left(\prod y_j^4\right).$$

Taking logs, reject $H_0$ when

$$\sum_{i=1}^{n}\ln x_i + 4\sum_{j=1}^{m}\ln y_j \ge c,$$

which matches choice **B**.
