A keyboard has 6 function keys, 4 arrow keys, and 2 control keys (12 keys in total). A quality control specialist randomly selects 3 keys for testing.

Calculate the expected number of function keys selected.

- A) 0.5
- B) 1.0
- C) 1.5
- D) 2.0
- E) 2.5

## Explanation

Let $F$ = number of function keys selected. By the **hypergeometric** mean formula (or linearity of expectation with $n=3$, $K=6$, $N=12$):

$$E[F] = n \cdot \frac{K}{N} = 3 \cdot \frac{6}{12} = 3 \cdot \frac{1}{2} = \frac{3}{2} = \mathbf{1.5}$$
