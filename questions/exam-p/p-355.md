Let $N$ be a Poisson random variable with cumulative distribution function $F$ such that $F(0) = 0.45$ and $F(1) = 0.90$.

Calculate $E[N^2]$.

- A) 0.90
- B) 1.00
- C) 1.45
- D) 2.00
- E) 2.45

## Explanation

$P(N=0) = e^{-\lambda} = 0.45$.

$P(N=1) = F(1)-F(0) = 0.45 = \lambda e^{-\lambda} = 0.45\lambda$, so $\lambda = 1$.

For $N \sim \text{Poisson}(1)$: $E[N] = 1$ and $\text{Var}(N) = 1$.

$$E[N^2] = \text{Var}(N) + E[N]^2 = 1+1 = \mathbf{2}$$
