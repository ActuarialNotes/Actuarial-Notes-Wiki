# Discrete Mathematics

## Definition

==Discrete mathematics== is the branch of mathematics dealing with countable, distinct structures including sets, logic, combinatorics, and graph theory.

> [!example]- <u>Example</u>
> **Set operations:** Let $A = \{1, 2, 3, 4\}$ and $B = \{3, 4, 5, 6\}$.
> - **Union:** $A \cup B = \{1, 2, 3, 4, 5, 6\}$
> - **Intersection:** $A \cap B = \{3, 4\}$
> - **Difference:** $A \setminus B = \{1, 2\}$
> - **Symmetric difference:** $A \triangle B = \{1, 2, 5, 6\}$
>
> These set operations form the foundation for defining events in probability theory, where events are sets of outcomes from a sample space.
