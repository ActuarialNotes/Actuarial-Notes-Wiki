Given the following countrywide information:

| | 2016 Expense Ratio | 2017 Expense Ratio | 2018 (\$000s) |
|---|---|---|---|
| Direct Premium Written | | | 6,100 |
| Direct Premium Earned | | | 5,920 |
| Commission and Brokerage Expenses Incurred | 12.0% | 13.0% | 945 |
| Other Acquisition Expense Incurred | 12.8% | 12.7% | 760 |
| General Expenses | 15.0% | 5.0% | 325 |
| Taxes, Licenses, & Fees Incurred | 2.1% | 2.2% | 130 |

- Profit provision = 7%.

## Part a (1.25 points)

Select and justify a total expense ratio for use in ratemaking assuming all expenses are variable.

### Explanation

First calculate the 2018 expense ratios:

- Commission and Brokerage (C/B): $945 / 6{,}100 = 15.5\%$ (variable, use DPW base)
- Other Acquisition (O/A): $760 / 6{,}100 = 12.5\%$ (variable, use DPW base)
- General Expenses (G/E): $325 / 5{,}920 = 5.5\%$ (use DPE base)
- Taxes, Licenses & Fees (TL&F): $130 / 6{,}100 = 2.1\%$

**Selection with justification:**

C/B jumped significantly in 2018 (15.5%) compared to 2016 (12.0%) and 2017 (13.0%). Average the three years: $(12.0\% + 13.0\% + 15.5\%) / 3 = 13.5\%$.

O/A has been stable. Average of three years: $(12.8\% + 12.7\% + 12.5\%) / 3 \approx 12.7\%$.

G/E dropped sharply after 2016. Use the average of 2017 and 2018: $(5.0\% + 5.5\%) / 2 = 5.5\%$ (suggesting 2016 was anomalous).

TL&F has been stable. Average of three years: $(2.1\% + 2.2\% + 2.1\%) / 3 \approx 2.1\%$.

$$\text{Total Selected Expense Ratio} = 13.5\% + 12.7\% + 5.5\% + 2.1\% = 33.8\%$$

(Alternative: use all 2018 ratios, giving total of 35.6%.)

### Examiner Report

Candidates were expected to calculate the 2018 expense ratios and compare/contrast them to the 2016 and 2017 values to justify their total selected expense ratios.

Common mistakes included:
- Using an average of 2016 and 2017 expense ratios directly without considering the given 2018 data.
- Using the incorrect premium base (written vs. earned) when calculating an expense ratio.

## Part b (0.25 points)

Calculate the variable permissible loss ratio using the expense ratio from Part a.

### Answer
59.2

### Explanation

$$\text{VPLR} = 1 - \text{Total Expense Ratio} - \text{Profit Provision}$$

$$\text{VPLR} = 1 - 33.8\% - 7\% = 59.2\%$$

(If using the 35.6% total expense ratio from Part a alternative: $1 - 35.6\% - 7\% = 57.4\%$.)

### Examiner Report

Candidates were expected to calculate the variable permissible loss ratio.

A common mistake was not considering the profit provision.

## Part c (0.5 points)

Briefly discuss two potential distortions from using the results in Part b to determine rates in a new state.

### Explanation

Any two of the following:

1. **Fixed vs. variable expenses**: If some expenses are truly fixed (do not vary with premium), using all-variable expense ratios will understate premiums for small-premium policies and overstate them for large-premium policies.

2. **Geographic variation**: Expenses such as TL&F and C/B often vary by state, so using countrywide ratios may not be appropriate for a new state.

3. **Historical vs. future expenses**: Historical expense ratios may differ significantly from future ratios, particularly if the book of business or market conditions change.

4. **Start-up costs**: Expenses will likely be higher while gaining traction in a new state due to infrastructure costs and new staff, making historical countrywide benchmarks too low.

### Examiner Report

Candidates were expected to provide examples of potential issues when using benchmark data in a new state or when assuming all expenses vary with premium.

A common mistake was not fully developing an answer — for example, simply stating there could be some fixed expenses without explaining how that would lead to distortion (understating premium for small policies, overstating for large policies).

## Part d (0.5 points)

Calculate the variable permissible loss ratio if 100% of the taxes, licenses, and fees and 75% of the general expenses do not vary by premium.

### Answer
65.4

### Explanation

If TL&F (2.1%) and 75% of General Expenses ($0.75 \times 5.5\% = 4.125\%$) are fixed, they should be excluded from the variable expense ratio and instead covered as fixed dollar amounts per policy.

The variable portion of expenses:

$$\text{Variable Expense Ratio} = 13.5\% + 12.7\% + (25\% \times 5.5\%) + 0\%$$

$$= 13.5\% + 12.7\% + 1.375\% = 27.575\%$$

$$\text{VPLR} = 1 - 27.575\% - 7\% \approx 65.4\%$$

Alternatively, starting from the Part b VPLR and adding back the fixed components:

$$\text{VPLR} = 59.2\% + 2.1\% + (75\% \times 5.5\%) = 59.2\% + 2.1\% + 4.125\% \approx 65.4\%$$

### Examiner Report

Candidates were expected to adjust the variable expense ratio for the fixed expenses and recalculate the VPLR.

Common mistakes included:
- Including 75% of the General Expenses as variable (instead of the stated 25%).
- Not considering the profit provision.
