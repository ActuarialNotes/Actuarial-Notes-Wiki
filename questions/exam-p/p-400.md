Let $J$ denote the number of small hailstorms per year, $J \sim \text{Poisson}(1)$. Let $K$ denote the number of large hailstorms, $K \sim \text{Poisson}(1/2)$. $J$ and $K$ are independent.

Calculate $E[J \mid J+K=1]$.

- A) $1/3$\n- B) $2/3$\n- C) $1$\n- D) $3/2$\n- E) $2$

## Explanation

$J+K=1$ means either $(J=1, K=0)$ or $(J=0, K=1)$.

$$P(J=1, K=0) = P(J=1)\,P(K=0) = e^{-1}\cdot e^{-1/2} = e^{-3/2}$$

$$P(J=0, K=1) = P(J=0)\,P(K=1) = e^{-1}\cdot\tfrac{1}{2}e^{-1/2} = \tfrac{1}{2}e^{-3/2}$$

$$P(J+K=1) = e^{-3/2}+\tfrac{1}{2}e^{-3/2} = \tfrac{3}{2}e^{-3/2}$$

$$E[J \mid J+K=1] = \frac{1\cdot e^{-3/2}+0\cdot\tfrac{1}{2}e^{-3/2}}{\tfrac{3}{2}e^{-3/2}} = \frac{1}{3/2} = \mathbf{\frac{2}{3}}$$
