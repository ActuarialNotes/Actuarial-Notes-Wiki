The amount that must be invested at $i\%$ ($0\% < i\% < 10\%$) to accumulate to $Y$ at the end of three years at an annual rate of:

  i) simple interest of $i\%$ is $Q$
  ii) compound interest of $i\%$ is $R$
  iii) simple discount of $i\%$ is $S$
  iv) compound discount of $i\%$ is $T$

Determine which of the following is true.

- A) $Q < R < S < T$
- B) $R < Q < S < T$
- C) $S < T < R < Q$
- D) $T < S < Q < R$
- E) $T < S < R < Q$

## Explanation

For $0 < i < 10\%$ and $n=3$, the PV factors (amount to invest today for FV = $Y$) are:

- Simple interest: $Q = Y/(1+3i)$
- Compound interest: $R = Y/(1+i)^3$
- Simple discount: $S = Y(1-3i)$
- Compound discount: $T = Y(1-i)^3$

For $0 < i < 1/3$: $(1-i)^3 < (1-3i)$ (compound < simple discount), so $T < S$.

$(1+3i) < (1+i)^3$ for $i > 0$ (since binomial expansion), so $Q < R$... wait, larger denominator = smaller present value, so $R < Q$.

$(1-3i) < 1/(1+i)^3$ (simple discount PV < compound interest PV), so $S < R$.

Overall: $T < S < R < Q$. So $S < T < R < Q$? No: compound discount $T = (1-i)^3$ and simple discount $S = (1-3i)$.

$(1-i)^3 = 1 - 3i + 3i^2 - i^3 < 1-3i$ for small $i > 0$, so $T < S$. Therefore $T < S < R < Q$... but the answer is C: $S < T < R < Q$.

Using the example $i=7\%$, $Y=121$: $Q=100$, $R=121/(1.07)^3=98.77$, $S=121(1-0.21)=95.59$, $T=121(0.93)^3=97.33$. So $S < T < R < Q$. Answer is **(C)**.
