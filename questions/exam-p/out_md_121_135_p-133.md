An insurance company insures red and green cars. An actuary compiles the following data:

| | Red | Green |
| :--- | :--- | :--- |
| Number insured | 300 | 700 |
| Probability an accident occurs | 0.10 | 0.05 |
| Probability that the claim exceeds the deductible, given an accident occurs from this group | 0.90 | 0.80 |

The actuary randomly picks a claim from all claims that exceed the deductible.

Calculate the probability that the claim is on a red car.

- A) 0.300
- B) 0.462
- C) 0.491
- D) 0.667
- E) 0.692

## Explanation

Let $R$ be the event the car is red and $G$ be the event the car is green.
$P(R) = 300 / 1000 = 0.3$.
$P(G) = 700 / 1000 = 0.7$.

Let $E$ be the event that a claim exceeds the deductible.
The probability that a red car has a claim exceeding the deductible is the product of the probability of an accident and the conditional probability given an accident:
$P(E \mid R) = P(\text{accident} \mid R) \times P(\text{exceeds} \mid \text{accident and } R) = 0.10(0.90) = 0.09$.

Similarly for green cars:
$P(E \mid G) = 0.05(0.80) = 0.04$.

Using Bayes' Theorem, the probability that the car is red given that the claim exceeds the deductible is:
$$P(R \mid E) = \frac{P(R)P(E \mid R)}{P(R)P(E \mid R) + P(G)P(E \mid G)} = \frac{0.3(0.09)}{0.3(0.09) + 0.7(0.04)}$$
$$= \frac{0.027}{0.027 + 0.028} = \frac{0.027}{0.055} \approx 0.4909$$
