You draw a large number of independent samples, each of size $n = 4$, from a uniform distribution on $(0, \theta)$. You want to use the second smallest value in each sample as an estimate for the mean.

The density for the $k^{\text{th}}$ order statistic of a sample is given as:

$$g_k(y_k) = \frac{n!}{(k-1)!(n-k)!}[F(y_k)]^{k-1}[1 - F(y_k)]^{n-k} f(y_k)$$

Calculate the expected bias of this estimate.

- A) $-\dfrac{4\theta}{5}$
- B) $-\dfrac{4\theta}{7}$
- C) $-\dfrac{\theta}{10}$
- D) $-\dfrac{\theta}{30}$
- E) The answer is not given by (A), (B), (C), or (D)

## Explanation

The second smallest value is the $k = 2$ order statistic, $Y_2$, of a sample of size $n = 4$ from $\text{Uniform}(0,\theta)$. The order statistics of a uniform distribution have

$$E[Y_k] = \frac{k}{n+1}\theta \implies E[Y_2] = \frac{2}{5}\theta.$$

The estimate targets the mean $\theta/2$, so the bias is

$$\text{Bias} = E[Y_2] - \frac{\theta}{2} = \frac{2\theta}{5} - \frac{\theta}{2} = \frac{4\theta - 5\theta}{10} = -\frac{\theta}{10}.$$

The answer is **C**.
