An actuary is using the inversion method to simulate a random number from a distribution with the following density function:

$$f(x) = 5x^4, \quad 0 < x < 1$$

A random draw of 0.6 was chosen from the uniform distribution $(0, 1)$.

Calculate the simulated random number.

- A) Less than 0.2
- B) At least 0.2, but less than 0.4
- C) At least 0.4, but less than 0.6
- D) At least 0.6, but less than 0.8
- E) At least 0.8

## Explanation

The **inversion method** sets $F(x) = u$ and solves for $x$. First find the CDF:

$$F(x) = \int_0^x 5t^4\, dt = x^5, \quad 0 < x < 1.$$

Set $F(x) = u = 0.6$:

$$x^5 = 0.6 \implies x = 0.6^{1/5} = 0.6^{0.2} \approx 0.9029.$$

This is at least $0.8$, so the answer is **E**.
