An annuity provides level payments of 1000 every six months for a fixed period.

Using an annual effective interest rate of $i$, the future value of this annuity at the time of the last payment is 19,549.25 and the present value of this annuity at the time of the first payment is 7,968.89.

Calculate $i$.

- A) 7.4%
- B) 8.5%
- C) 15.4%
- D) 17.0%
- E) 17.7%

## Explanation

Let $n$ = number of payments, $j$ = semiannual rate. The FV at last payment and PV at first payment are $n-1$ half-years apart:
$$7,968.89(1+j)^{n-1} = 19,549.25$$

Also $7,968.89 = 1000\ddot{a}_{\overline{n}|j} = 1000(a_{\overline{n-1}|j} + 1)$, so $a_{\overline{n-1}|j} = 6.96889$.

Solving: $j = \dfrac{1 - 7,968.89/19,549.25}{7,968.89/19,549.25 - 1} \cdot \dfrac{1}{7,968.89/1000 - 1}$... 

Using the relation $7,968.89(1+j)^{n-1} = 19,549.25$:
$$j = \frac{1 - 7,968.89/19,549.25}{7,968.89/1000 - 1} = \frac{1-0.40763}{6.96889} = \frac{0.59237}{6.96889} = 0.085$$

Annual rate: $i = (1.085)^2 - 1 = 0.1772 = 17.7\%$
