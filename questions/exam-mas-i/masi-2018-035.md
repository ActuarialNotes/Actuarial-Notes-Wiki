You are given the following three statistical learning tools:

- I. Cluster Analysis
- II. Logistic Regression
- III. Ridge Regression

Determine which of the above are examples of supervised learning.

- A) None are examples of supervised learning
- B) I and II only
- C) I and III only
- D) II and III only
- E) The answer is not given by (A), (B), (C), or (D)

## Explanation

**Supervised** learning uses a labeled response variable; **unsupervised** learning seeks structure in predictors without a response.

- **I. Cluster Analysis** — unsupervised (no response variable; it groups observations).
- **II. Logistic Regression** — supervised (predicts a categorical response).
- **III. Ridge Regression** — supervised (predicts a continuous response with a penalized fit).

II and III are supervised, so the answer is **D**.
